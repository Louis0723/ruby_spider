「大家都想要一帆風順的愛情，我也老實說，人際關係中沒有一帆風順，不然就等於彼此之間沒有任何交集。就像是有張梗圖：『我的愛情路十分順暢，因為一路上都沒有人。 』」

在YouTuber 米鹿 的頻道上，擁有人類圖官方一階認證的 Repeat 老師受邀解答大眾對戀愛的各種問題，淺顯易懂的解釋，巧妙又生動的比喻，讓人類圖不再令人感到艱澀複雜，反而更能貼近生活，至今影片觀看數已衝破十多萬次。

超過 780 小時的授課、定期在 Clubhouse 上與大眾的互動，喜愛與人互動的特質，就像是 Repeat 的稱號一樣，開著這台「人類圖計程車」到處與人交流，在彼此的人生中，相互載送一程。

而這些積累，也讓 Repeat 不僅精通人類圖背後的理論，還能回到生活情境，點出行為背後是源自哪些特質的相互干擾。讓人類圖這門「區分的科學」，不再是艱澀難懂的定義，而是能在生活中，不斷地被實驗的科學。

「我其實很少下指導棋，這樣是在限制可能性。告訴對方，行為背後的原理是如何運作的，反而能幫助他拿回人生的主導權。」

Repeat 分享，所有人際關係，尤其是愛情，都得從「認識自己」開始。而透過人類圖這本「人生使用說明書」，了解自己行為背後的成因， 理解自己、擁抱自己，才能回到關係中，找到兩人之間的平衡。

踏入關係前，先搞懂我是誰？我想要的是什麼？

Repeat 收到最多的問題就是「如何在戀愛中與對方相處？」。交往中的想確認問題出在誰身上，交往前、曖昧中的則是想知道如何跟對方相處，好讓他能更愛我。

然而，在「愛與相處」的課題中，第一步其實是「認識自己」。

「你一定要先知道自己是誰，才有辦法跟另一個人相處。

如果不知道自己是誰，很容易活在別人的投射、期望裡。」

Repeat 提到，自己無意識地調整成對方喜歡的模樣，反而會壓抑真實的自我。在關係中感到不舒服、不斷積累後，總有一天會爆發，營造出來的人設，終究會破滅。

當「對方的期待」與「自己的喜好」之間的界線模糊不清，其實就是所謂的「人我分際不清」。在這個情況下，很容易會怪罪對方「我為你付出了這麼多，為何你還是不喜歡我？」其實關係的天秤在一開始，就已經逐漸走向失衡。

也許我們會開始反省自己，但 Repeat 卻說「不自覺想符合社會、他人期待的樣子」的行為並非你的錯，這其實是大多數人天生的能量場設計。

舉例來說，幾乎佔了總人口數的七成以上的「生產者」，他們的能量場是完全開放的。也就是說容易受到他人的期望、情緒影響。若是沒有自覺，便很容易成為社會、他人期待的樣子，而背離自己原本的樣貌。

Repeat 談到在進入關係前，「生產者需要被問，因為你真的不知道吸引自己的究竟是什麼？」舉個例子，生產者很容易被一個人吸引目光，就開始腦補各種情節。

這時，若有人問他「你是不是覺得他身上的衣服好看？」生產者可能會直覺反應回答「恩！」但要是再問「他追你的話，你有要跟他在一起嗎？」生產者可能就沒有「恩！」的回應了。

原來，生產者自己也不知道，吸引自己的並非這個人，而是他的氣質、衣著甚至是那天剛好風和日麗的氛圍。通過正確的問答，才能明白自己的能量如何流動，進而找到適合自己的人。

透過了解自己是哪種類型的人，運用人類圖中對應的方式幫助自己設下界線，釐清「我是誰，你是誰，我們中間的界線在哪裡？」。搞懂自己真正想要的是什麼，才能在關係中找到平衡。

找到彼此的界線，才能改善關係中的摩擦

「在相處的課題中，我覺得『尊重』是一件很重要的事情，找到彼此的界線才能相互理解、尊重彼此。」Repeat 分享，要先知道自己是什麼樣子，才會了解對方與自己的不同之處，而知道彼此差異後，因著相愛，不會硬要改變對方，這就是一種「尊重」。

看了這麼多對情侶，Repeat 發現「很多摩擦其實不需要累積，只是一開始因著不了解而產生誤會。」

舉例來說，有一對情侶，其中一人的人生角色是需要自我空間的 3 爻，另一人是重視緊密關係的 4 爻。兩人看待吵架的觀點便不同，對 4 爻人來說，這是一種連結，每次吵架都是一種溝通，感情的付出又更深刻了。

但是對 3 爻人來說，一次次的吵架卻成了變相的壓力，是不適合的證明，既然不適合，那就切斷關係好了。

「我們常常會用自己的方式去理解對方，會覺得這樣就是對他好，我用我自己的方式照顧他，但那可能不是他要的，因為他並不是那樣看事情的啊！」

Repeat 認為人類圖在關係中就像是一個「路標」，設立一個基準點，在遇到情況時，能讓兩人察覺彼此想法上的差異，進而相互理解。

像是當我們知道對方是 3 爻人後，就能了解 3 爻天生設計就需要個人空間，他並非心裡有別人才保持距離，於是不必要的腦補就可以減少。理解要給對方空間並非關係變得淡薄，也不是「你要空間，我都給你！」這種反而讓 3 爻人壓力山大的體諒。

「當你們理解彼此，才有辦法做出不一樣的選擇，去改善你們的關係。」人類圖作為感情上的路標，在遇到情況的時候，能讓我們更有機會正確地與彼此互動，不讓當初吸引彼此的優點，成了相處時的阻礙。

找到問題的施力點，才能更好地做出選擇

「我會講解原因給你聽，給你一個路標，但你的人生一定要有自己的選擇權。」

如果你曾自行研究人類圖或是星座，你可能會注意到，某些人的解釋，會讓你覺得很不準，但另一個人的描述，卻讓你很有感。

一直以來，Repeat 老師都傾向以簡單的方式講解人類圖背後的運作方式，再以生動的方式舉例說明。帶學生們認識自己的原生設計，才能在下一次遇到相同狀況時，有意識地選擇不同的解決方式。

人類圖像是人生裡的路標，在滿滿地無力感中，指出癥結點，透過 Repeat 老師的解說，找到問題的施力點，還你人生的選擇權！

曾有一位學生，他的人類圖中有 55 號閘門，在戀情中，他常常不知道如何表達自己，最終對方便會十分不耐地以「你現在想什麼，講出來啊！」、「想講什麼就講！為什麼不講？」的方式溝通，但她依舊不知道該如何表達。

在人類圖的原文定義中，55 號閘門是「在不同維度當中散落的情緒」，翻成中文之後，依舊感到似懂非懂。也很常被伴隨著一個標籤「要找靈魂伴侶」。但是誰不想找靈魂伴侶呢？看到此，可能就會覺得不準啦！

這時 Repeat 老師解釋，其實背後都是有原因的，有 55 號閘門的人，情緒十分豐沛，這種豐沛並非瀑布一瀉而下，而是各種情緒都有一點點，導致他們無法完整表達。

而為什麼要找靈魂伴侶？這是因為他們說不出口的情緒，需要有一個人耐心地詢問他們「你是不是……，這樣想？」。Repeat 形容就像是在擠牙膏，一點一點慢慢來。當對方能幫他說出心裡的感受，讓他有「對！就是這樣！」就會是他們需要的靈魂伴侶。

在 Repeat 老師詳細地講解後，這位學生才了解自己在溝通上的特點。不再過度責怪自己「表達能力太差」、「不會溝通」的缺點。

在下一段關係中，他也先坦承自己的情況，發現對方願意與他一起學習。當一個人願意多講一點，另一個人願意多問一點，雙方開始互動了，逐漸產生對話，他才有辦法表達出東西。

快上車，讓「人類圖計程車」載你一程！

「其實我是希望自己可以，真的可以載大家一程。」談到幫助學員的喜悅， Repeat 解釋自己人類圖中有一條 2-14 通道，又被稱作「計程車司機的通道」，意思是人生的走向難以自己決定，就像是計程車司機一樣，永遠不知道自己今天會載到誰？又會載去哪裡？

「因為人類圖是我有回應的事情，所以當我走在正確的路上，大家就能在我的能量場中獲得啟發、力量，自己得到人生方向，我覺得這樣是非常好的事情。」這條計程車通道載著 Repeat 繼續往前去認識更多人，而薦骨中心的設計，讓他的能量場無形中能影響他人，進而找回自己、得到啟發。

於是，Repeat 的人類圖計程車即將開跑！這次將著重在「愛與相處」的課題，不論你是想認識自己、想了解另一半究竟在想什麼，還是在關係中老是遇到一樣的問題。

快上車，與 Repeat 聊聊你的煩惱！
