如果說，個人電腦帶來的資訊革命，開啟了我們所處的數位時代。那麼 Apple 的麥金塔與 Microsoft 的 Windows 可以說是，賦予你我「不必懂程式語言也可以輕鬆使用電腦」便利性的起點。

但這項里程碑，卻不是賈伯斯與比爾蓋茲的原創，而是藉由模仿後的再超越所做到的。在《逆向工程，你我都能變優秀的秘訣》書中，以麥金塔與 Windows 的起源故事，帶出商戰上公開的秘密：贏家通常擅於藉由「逆向工程」解構成功秘方。

Apple、Microsoft 的巨大成功，都運用了「逆向工程」

1979年，全錄公司想要投資蘋果，賈伯斯藉機要求參觀全錄當時最先進的帕羅奧圖研究中心，想要一探全錄的研發實力。

當時同行的除了蘋果的工程師之外，還包括當時替蘋果開發應用程式的比爾蓋茲。

當年的電腦，要靠鍵盤輸入程式語言選項操作，還沒有現在的滑鼠游標。因此，當他們看到帕羅奧圖研究中心裡，研發的滑鼠與圖像操作介面原型機時，賈伯斯眼睛都亮了。

賈伯斯決定要偷這個概念來做出新世代的電腦！他曾引用畢卡索的話「好的藝術家懂得模仿，偉大的藝術家則善於偷」。

賈伯斯認為所謂的模仿比較像照抄，但「偷取概念」則是在改良點子，甚至是超越原來的想法。他覺得自己可以在全錄這個精彩的概念之上，做出更好用的電腦，那就是後來的麥金塔（簡稱為 Mac  ）。

微軟後來也以同樣的概念開發了 Windows，賈伯斯一直覺得比爾蓋茲是抄襲自己的想法。

比爾蓋茲則說：「嘿，Steve，我們兩個其實就像是一起翻牆，進到全錄這個有錢鄰居家偷東西，剛好都看中了那台電視，只是你比我快了一點出手而已。」

比爾蓋茲的意思是，賈伯斯的麥金塔也不是原創，所以不要覺得我是抄襲你的。

這個故事還有後續，多年之後，微軟先推出了平板電腦 ( Tablet PC )，這次換成賈伯斯模仿這個概念且又超越。

他以更方便的「觸控操作」取代 Tablet PC 的手寫筆，開創了 iPhone 與 iPad 這兩項劃時代的產品。

從 Apple 與 Microsoft 這兩大巨頭的競爭中我們可以看到：成功的秘方不一定要靠完全原創，而是優秀的洞察與執行力。

不只是資訊產業，汽車業、製藥、餐飲…，各行各業裡都可以看到藉由逆向工程解構成功企業的方法藉此做出突破的成果。仍然要強調，逆向工程並不是鼓勵剽竊與抄襲，否則也不必寫本書來教了。

想做到先模仿再超越？

要先學會這 3 招

該怎麼解構出別人的成功方程式，再設法調整並超越呢？這絕對不是依樣畫葫蘆這麼簡單而已。

在這裡先借用金庸小說《倚天屠龍記》裡的一段情節來比喻：因為臨敵需要，張三豐必須馬上教會張無忌《太極劍法》。

他總共示範了三次，每次的招式都不同，直到張無忌說「招式我全忘了」，這時張三豐居然點頭道：「那你可以去和他們過招了」

之所以張三豐用三次全然不同的招式教張無忌太極劍，甚至可以要讓張無忌忘記招式之後再去比劍，是因為他要傳授的不是「劍招」（技法），而是「劍意」（模式）。

在《逆向工程，你我都能變優秀的秘訣》書中這樣形容：

「要看出成功的範例中共通的隱藏結構，你必須稍微後退一步遠觀。」

「模式」就是我們藉由逆向工程拆解之後要學會的。在這裡，將介紹 3 種方法，製作逆向大綱、設計自己的計分板、具體想像過程。

第 1 招：製作逆向大綱

若想要提升在眾人面前簡報的能力，可以藉由觀摩其他人的簡報過程，從中學習。但是成功地模仿，絕對不是仿效別人的話術或是語調，而是藉由拆解出「逆向大綱」來分析模式。可以怎麼做呢？

假設我們以TED的簡報影片做為學習的對象，可以嘗試從以下方向，拆解那些你認為精彩絕倫的簡報，並做成大綱，像是：

  • 長度：簡報進行的時間與內容字數
  • 結構：整場簡報裡，開場白佔多少比重？分成幾個論點？結尾佔多少？
  • 內容：有說故事嗎？有安插笑話嗎？插在哪些段落？事實佐證用的例子有幾個？句子長度大約多長？
  • 情緒：正面情緒、負面情緒、中立情緒的佔比？
  • 表達技巧：說話速度每分鐘多少字？有什麼肢體語言？會來回走動嗎？
  • 投影片製作：每分鐘講幾張？每張上面大約多少字？有用圖片嗎？圖片張數和形態？

在拆解足夠多的案例之後，以這份大綱來設計簡報，就能達成近似的精彩效果了。

第 2 招：設計自己的記分板

當然，好還要更好，如何設法持續提升我們的表現呢？

最明確的方式是將表現「指標化」，要先找出重要的指標項目，再運用數據做量化管理。在這裡借《亞馬遜逆向工作法》書中 Amazon 的案例來說明：

電商的目標在於成交與銷售金額，初期 Amazon 認為商品選擇越多，成交的機會越大，因此將「商品上架種數」視為指標。

商品部門努力地增加在 Amazon 上可以銷售的商品種類，但是隨著商品種類數的上升，銷售額似乎並沒有同步增長。這是為什麼呢？

當增加的商品顧客並沒有興趣，根本就不會帶來成交量。因此 Amazon 將指標調整為「商品頁的瀏覽量」，並用各種方式讓到訪的顧客能看到更多吸引他的商品，唯有消費者瀏覽到商品頁，才有成交的機會。

之後這項指標還會持續的優化，提高成功銷售的機會。這就是藉由指標優化表現的方式。

在《逆向工程，你我都能變優秀的秘訣》裡提到：設計屬於自己的記分板。如果要衡量我們的績效、有效地運用指標，以下是我們可以進行的做法：

試著將單一活動拆解成多項「次級技能」，分別紀錄並衡量自己的表現。

接續前面簡報技巧的例子，可以把「做出吸引人聆聽的簡報」拆解成以下技能的總和：

1.內容精彩度

2.表達技巧

3.投影片製作能力

針對這些項目，可以由自己、同事或朋友來幫我們的表現評分。最終，便能發展出最佳的整體表現、具有自身特色與優勢的風格。

第 3 招：具體想像過程

除了藉由每一次的實戰表現來衡量、優化自己表現之外，我們還可以藉由「在想像中預演」幫我們的表現加分。

就像是不同領域的優秀運動員都共有的習慣：在腦中模擬自己在練習的情境，以及想像比賽的過程。

之所以要做「想像的練習」，是因為從各項研究中都發現：即便只是想像的練習，都對表現有實質助益，原因有以下幾點：

1.辨識出可能發生的障礙：在腦海裡模擬簡報，可以發現自己在哪幾張的連接、轉場可能不順，進而做出調整。

2.提供情緒預告：在想像的練習裡，我們會漸漸習慣可能會有的緊張情緒，甚至是模擬出錯後的應變方式，避免實際進行時束手無策

3.預先做出應變：已經在腦中排練出錯的情境，就可以在發生前先做好準備

4.提升自信：在想像練習裡，完美的表現或精彩的應變，可以幫助我們增強自信心。

5.啟動大腦神經反應：雖然只是想像，但因為大腦不斷地產生與實際進行時相同的神經反應，因此讓我們的表現更熟練。

在這個社群媒體與內容蓬勃發展的時代，我們可以發掘、認識到無窮的成功個體、案例。

「逆向工程」的技法，會是一項非常有用的技能。可以幫我們從其他人的成功中拆解他的成功秘方，並且轉化成我們的能力。這也是《逆向工程，你我都能變優秀的秘訣》要教會我們的方法。
