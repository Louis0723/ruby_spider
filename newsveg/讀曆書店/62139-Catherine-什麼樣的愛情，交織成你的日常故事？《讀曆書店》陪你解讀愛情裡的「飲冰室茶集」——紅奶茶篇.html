在愛與被愛之間，我們持續摸索愛情的全貌。
每 3 分鐘檢查一次手機，想知道他回訊息了沒。
這週六約她看電影，不知道會不會被拒絕。
他知道其實我不想只做朋友嗎？
這或許是成千上萬個他與她的現在進行式，你的愛情故事現在走到哪個階段？
一天 24 個小時，我們工作、購物、聚會。而愛情，它不說一聲就闖進你我的生活，卻又可能突然消聲匿跡。如果真要給生活一個定義，那就是愛情與忙碌交織成的日常故事。
「讀曆書店」與「飲冰室茶集」合作，在紅奶茶的包裝上，收錄 7 位作家的愛情選句。透過作家介紹與他們的愛情刻畫，我們一起摸索愛情的模樣。

陳依文《萌》／心靈工坊

戀愛了，一條條原則 / 被愛情炸過 /
擱浸在鮮甜多料的生活裡 / 吸飽湯汁 泡得發軟了


考上台大電機系，卻沒有因此走入理工這塊領域，陳依文反而在文學的世界裡，拼湊出一條屬於自己的路。轉中文系後，她繼續攻讀台大台文所，與文字有了更深的羈絆。
這篇選句出自詩集《萌》，道出愛情裡的人們，對於愛情的束手無策。原則可以被改變，生理與心理，都可以因為攝取了幸福，而變得軟綿甜蜜。

少女老王《那一年，那些沒人說的故事》／圓神出版

我是因為遇到了妳才知道，喜歡上一個人，本來就該全力以赴。


媒體人、上班族的日常該是什麼模樣？少女老王提筆寫下她在職場、社會所看見的百態。有我們天天都看到的職場文化，也有身為女性會碰到的性別歧視。也許不滿，或許憤慨，但這些情緒都會在少女老王的筆下，化作柔軟的畫面。
或許是出自媒體人的本性，少女老王下筆雖然溫柔，但還是能夠透過文字，看出她直言、善於觀察細節的性格。在《那一年，那些沒人說的故事》中，少女老王不只帶領讀者觀察疫情下的世態炎涼，也捕捉了無數溫暖與感動，將這些人性百態收錄在書中。

李國祥 《伸縮自如的愛》／印刻文學

你是我的填充題 / 我是你的選擇題 / 我們的關係遂成了申論題


中文系畢業、擔任出版社編輯的李國祥，將日常生活中遇到的苦悶、厭世、無解，以平易近人又諷刺的語氣，替同樣是平凡人的讀者們，宣洩身為社畜的無奈。
李國祥將厭世文學，以短篇詩作的方式處理。儘管文字承載著對生活的煩悶，但也因為詩文的輕薄短小，而不讓人讀來感到沉重。除了描繪無奈的日常，也有像此篇選句一樣，讀起來有點繞、有點酸澀，也有點奇趣的情感詩篇。

林婉瑜《我沒有談的那場戀愛》／尖端出版

你對我笑的時候總是那麼真實，沒有暗色塊，沒有糾結的細節在神情裡。


進入臺北醫學大學後休學，林婉瑜最後畢業於臺北藝術大學戲劇系。大學時主修劇本創作，但她和讀者最親密的接觸，是透過詩作。出版多本詩集，林婉瑜擅長在生活中捕捉體悟，她寫下愛、人性、生命、世界，將之譜成詩作。
《我沒有談的那場戀愛》是林婉瑜第一本文集，有別於詩的點到為止，要讀者親自體會。在這本書中，更加直敘了對情感的想法，寫下愛情的酸甜、感傷、釋懷，以及最後的忠於自我。

湖南蟲 《最靠近黑洞的星星》／時報出版

不要問我為什麼喜歡你 / 隕石撞不撞地球 / 有原因嗎？


並非文學院出身，但湖南蟲獲得不少文學獎項，包括林榮三文學獎、聯合報文學獎、時報文學獎。他透過散文與詩集，幫助讀者訴說，也透過文句剖析自我。
這篇情詩收錄在《最靠近黑洞的星星》，道盡愛情裡付出比較多、喜歡比較多的那人，以及告白之人的心聲。有無可奈何，也有愛情裡最直率真摯的情感。你可以說他有點橫衝直撞，但你也可以看見他毫無保留的真心。

宇路《星系明體》／奇異果文創

我渴望擁抱 / 髒了也沒關係


從台大社會學系畢業，到東華大學攻讀華文所，本名郭申睿的宇路把情感揉進文字，拾起生活與愛情中的體悟，再加以文字雕琢。透過他對於事物、空間細膩又奇幻的敘事方式，我們咀嚼愛情絮語。
「我渴望擁抱 / 髒了也沒關係」摘自詩集《星系明體》。在這本詩集中，筆下的愛情與星空結合，讀者可以踩著宇路的詩句，在宇宙和情感中漫步。

【特別收錄】鯨向海《每天都在膨脹》／大塊文化

黑暗之心中 / 對彼此的想念 / 是唯一發光體

 
詩人與精神科醫師，鯨向海的雙重身分看似沒有交集，但從文字中就可以明瞭，兩種身份反而讓他能同時以理性，又溫柔的方式下筆。鯨向海的文筆不華麗、不繾綣，把詩看得平常，也寫出平常人會看的詩，這就是他想呈現的日常。
「黑暗之心中 / 對彼此的想念 / 是唯一發光體」選自《每天都在膨脹》－ 鯨向海的第六本詩集。以最直觀、最貼近生活的用語，寫出對於相愛的人們來説，「思念」能夠在面對未知與困境時，照亮對方、給予彼此勇氣。
