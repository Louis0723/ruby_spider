前一陣子發生幾件家長「體罰」孩子，甚至已經是「過度管教」或「兒虐」的案子（打耳光 50 下／差一分打一下），看了真的很讓人難過。也許剛好是跟大家分享這篇文章的時候。

這篇文章的許多內容，特別是小時候的記憶，是半年多前就寫好的！寫的時候，不知道為什麼的－我邊寫邊流淚（在高鐵上），某個部份，這也像是一種自我挖掘，把過去的傷口打開，讓自己有機會讓傷口復原，而不只是埋在深深的記憶中。也希望我自己的經歷能鼓勵大家，雖然不容易，但我們總是可以越來越好，試著讓過去的錯誤，停留在我們身上就好。愛孩子，就別打小孩！

昨天早上我特別問了2個女兒：「爸爸有打過你們嗎？」。兩個女兒搖頭說沒有，說「有時我們會打爸爸 ….」（在玩的時候）。事實上，我的脾氣不好，耐心不佳。雖然在斷線時刻，也會發怒大聲。但 … 我從來沒打過2個寶貝女兒！

失控的脾氣

在大部份時候，我都表現的很正常，工作、生活、陪伴小孩、運動、專注的把手邊的事情做好。有時我覺得這幾年的我，像一個調較良好的生活戰士，朝向工作與生活的目標前進。而跟孩子在一起時，又能切換回到溫暖好爸爸模式，陪著孩子們成長 … 。一似看起來似乎都很美好 ….。

但是，我發現當我脾氣上來時，我似乎變成另一個人，另一個我快要無法控制的綠巨人。有時跟老婆吵架，或是脾氣上來的時候 … 這時心中會有很多的怒氣、講話開始大聲、言語開始刻薄，甚至大吼。在杏仁核作用下，就像是一頭生氣的熊！孩子那時也會離我遠一點，自己去書房或房間玩 …。雖然我不會（也不曾）因此體罰孩子或傷害家人，但怒氣上來的時候，我感覺自己像是另一個人 … 沒有平常的冷靜、精準。有時甚至會「生氣自己為什麼生氣」，但脾氣還是不受控。一直到某一個時刻，怒氣消失 … 我才會變回平常樣子 ….。

隨著這些年的成長，自己也曾想過，這麼大的怒氣 … 是怎麼來的？為什麼怒氣一上來時，自己像是無法控制？這是我希望的樣子嗎？雖然大部份的時間，我都表現的很好（自己覺得啦），老婆／孩子也很愛我。但是為什麼脾氣一上來時，我卻不容易控制它，真的是綠巨人嗎？為什麼有很多的氣在身體裡，而我似乎快把自己漲破了 … 。

我隱約的感受到有些事情 … 是在我小時候就開始。

對爸爸的記憶

小時候我們家開電器行，家裡面擺滿了電視跟音響，在接近 50 年前，在沙鹿老街 …這是很厲害的事情啊！那時還流行熬夜看少棒，或是晚上看三台節目，在我模糊的印象中，鄰居街坊都會來我家裡擺著椅子，一起看電視，我們家就像社區街坊的交流中心！身為長子＆長孫（上面還有2個姐姐，弟弟在我7歲時才出生），我備受寵愛。

印象裡的爸爸就是愛家、努力、不煙不酒的人，早期沒有第四台，都是用天線接收電視訊號，每到颱風天，許多電視天線會傾倒損壞，爸爸就會帶著我，去客人家修電視天線。有時候他也會帶我去海邊玩，甚至在很小的時候我們老家屋頂上面竟然用磚蓋了一個小游泳池，爸爸會帶我爬上屋頂泡水，後來爸爸也在裡面養蝦子（這是真的！）。家裡小康，雖然不富裕，但那一堆電視、夠讓小時候的我覺得很厲害了！

爸爸也很會彈吉他，只要哼一首歌出來，就能馬上伴奏；也記得他在桃園工廠當總務經理時，我常常趁著暑假去工廠玩，在工廠宿舍一住就是1～2個禮拜。爸爸似乎也會一點武術，小時候就跟我說：以後可以去學空手道或合氣道，也許也是因這樣，後來才影響了我去學合氣道，今年（53歲）開始學空手道。家裡面也有一整個書櫃，印象裡的我好像在國小時，就拿爸爸書櫃裡的《雅舍小品》來看（梁實秋），所以他應該也喜歡看書吧？（那時年紀小無法判斷）。大部份的時候，爸爸都對我很好，雖然臉色還是帶著威嚴，但是對我還是很疼愛的。

但是爸爸生氣的時候，會打人！以現在的說法，是一個會家暴的爸爸！（在天上的爸爸，Sorry … 請原諒我提到這件事 …）

被打的回憶

記得國小1～2年級，我吵著要買錶（細節我真的不記得了，後來問二姐，說我看到她買錶，我也想要，然後吵鬧)。結果我就被爸爸修理了！他拿著家裡的電視訊號線（扁的），朝我身上及腿上狠狠的抽下去，我記得我大哭、卻又跑不掉，後來是奶奶擋我身前，才停止被打。我隱約的記得家人把我帶到鄰居家，然後腳上被打的一條一條的傷痕，很久才好 … .

也記得有一次爸爸跟家人吵架（是阿公或是媽媽？我那時真的太小搞不懂 …），他怒氣沖天的拿起鐵鎚，把家裡的好多台電視螢幕砸爛！（快50年，那真的很珍貴！），在很淡的印象中，我記得媽媽或奶奶帶我們，躲到鄰居家 …

爸爸在暴怒失控時，也會打媽媽 … 小時候我不懂，也只會害怕，但隨著年紀越來越大，好像是在國中的某一次，爸爸又要動手時，我站在媽媽前面跟爸爸說「不準你動手打人！」。記得那是我第一次保護媽媽 … 從那天開始，我就不記得還有發生過類似的事了。

最後一次被爸爸打，應該是在五專時，那時我已經長高到 175 cm 了吧？！已經比爸爸還高了。有一陣子台灣流行六合彩，記得是因為勸爸爸不要再簽賭，不要浪費錢 … 然後引發他大怒！他在大街上朝我揮拳，用拳背打到我的臉，打到我嘴角流血 … 。那時我沒有哭 … 只有瞪大的雙眼，還有滿臉的恨意！「勸你不要賭博，你竟然會因為在大街上這樣打我 …！我會記住一輩子的 ….」。

身為爸爸之後

事情經過了30多年了，爸爸也已經到天上了 …我還記得那天在大街的場景。不過 … 只是記得，沒有恨意。

在大街打我的事件後沒幾年，爸爸就中風了！中風之後，他的威嚴消失了，只是一個不良於行的爸爸 … 。再沒幾年後，因為腎臟的併發症，他開始洗腎，就變的更虛弱了。之後，就在我進入職場的早期，爸爸就離世了。

這幾年我自己成為爸爸，有時候自己也會想 … 爸爸，也是開始當爸爸之後，才開始學習怎麼當爸爸吧？因為想寫這篇文章，我特別去問了二姐小時候的事。才知道那一次因為買錶被打的腳上一痕一痕，哭著睡著了 …. 。然後二姐看到爸爸進到房間裡，幫我雙腳的傷痕抹藥，她在房門外偷看，看到爸爸臉上滿是後悔及心疼 … 。現在的我會想，我知道爸爸愛我，但是在失控時，找不到方法跟出口。當杏仁核開始主導，而造成無法控制的大怒時，事後的爸爸 … 一定也很後悔吧。

自己的功課

現在我53歲，都已經快比爸爸離世時還老了。兒時的回憶及經歷，當然對我也會有一些影響，不管好的、壞的，都形塑成現在的我。寫這篇文章時，又想到一些事 …雖然沒辦法記得太仔細，但眼淚卻常常流下來。我知道自己小時候也曾受過傷，甚至記在心裡很深的地方。但，所有的不對，到我身上為止就好。我已經足夠強大，強大到可以從傷痕中復原。

雖然我還是有我自己的功課要做，譬如控制不了的脾氣，或是變成綠巨人般的怒氣。有時我脾氣上來，似乎快要變成爸爸的樣子了 ….。但我跟爸爸的事，到我身上就好。我跟爸爸不一樣，我有學習、有醒思、有成長。我身上有爸爸的基因，但卻不需要複製爸爸的方法！

所以，我絕不動手，不打小孩！不傷害家人！這不是選項！

成績是短暫的、情感才是永遠的

對了，雖然我在天上的爸爸，成長的過程體罰過我很多次。但他從來沒有因為成績的問題打過我！媽媽從小到大，也從沒有因為成績的問題念過我，更不要說打我。（我媽媽不體罰，謝謝媽媽，媽媽我愛你）

雖然我國中成績爛，五專也平平，大學沒有念。但我自己一直想學習，也樂在學習。沒有爸媽的「鞭打」，最終也寫了8本書 + 1 本博論，拿了 MIS Ph.D & EMBA ….. 。所以拜託，請不要拿成績為理由，用「為你好」為藉口。去打小孩好嗎？！

現在我53歲了，回想國小＆國中被體罰的記憶，包含爸爸及學校的老師們。我不記得任何他們想教我的事，只記得我被打 ….！除了疼痛，還有不甘心。那時只是因為我是小孩，我是學生。所以逃離不了 …。如果回到過去，我會站在小時候的我前面，保護他，不讓他被打。

小結

每個人都有自己要面對的功課，不管是上一代的傳承，或小時候的管教及不當的對待。但我們自己長大了，成人了 … 想辦法不要變成自己討厭的人，好嗎？

當然，這也是一件不容易的功課。我的爸爸修了一輩子，最後是因為生病才改變了個性。而我們現在有更多的資訊、更多的學習、更多的磨練、更多的愛。我們想辦法，讓自己變的更好，讓一切的不好－到我們身上為止。好嗎？

我也持續在努力，一起加油。
