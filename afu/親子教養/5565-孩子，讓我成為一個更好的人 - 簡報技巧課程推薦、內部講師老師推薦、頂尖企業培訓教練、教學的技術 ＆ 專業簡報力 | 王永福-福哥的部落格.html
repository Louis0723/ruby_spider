連續兩個禮拜帶著女兒跟我一起去《遊戲化教學的技術》演講，上週末去澎湖，這週則是去花蓮。兩個寶貝都當了很稱職的攝影小幫手。在演講結束時，也坐在桌旁跟我一起幫讀者簽名，甚至在演講結束後，也跟我一起跟主辦的好朋友／老師們一起吃飯。可以說是在第一線跟著爸爸一起工作！（爸爸去哪裡）。帶著孩子一起，雖然行動速度跟準備需要額外多一些時間，但是滿足感卻不曉得多了幾倍！看著寶貝們跟我一起站在台上，我想這樣的體驗 … 應該是最好的身教吧？！

在澎湖場演講結束後，跟啟民主任及澎湖教學輔導團的老師們一起聚餐，老師們在看到我跟女兒相處的樣子，以及云寶沈穩的表現後。有老師問：「福哥的年紀（我52歲），孩子還蠻小的（老大 11 歲），是因為比較晚結婚嗎？」

我停了一下，小小聲的說：「真正的原因 …. 是因為原本沒計劃生小孩 …..」哈！（小小聲怕云寶聽到 )

這其實是放在心底的一個小秘密！但最近遇到一個朋友，也是聊到孩子的議題，他們跟我以前一樣，沒計劃生小孩 … 。所以很好奇其中的轉變，以及有了孩子之後的改變。我覺得用說的怕不夠精準，就請他等著看我的文章。

先說結論：「孩子，讓我成為更好的人！」。然後，給我親愛的朋友，「如果沒有孩子，你真的不曉得你這一生錯過了什麼！」。

以前的我們

大約在 36 歲之前，我跟老婆還是想說以後可能就2個人過一生了。原因當然有很多，簡單的2句話就是「不敢生」＆「不想生」。不敢的原因有很多，某部份是覺得生命是一個很重大的責任，生了一個孩子 … 不知道能不能給他最好的未來。那時工作面臨轉變期，連自己的未來是怎麼樣 … 都不是很清楚了，更何況是給孩子一個未來；而不想的原因是：2個人相處習慣了，也很自由自在。那時養了一隻狗，平常有狗陪伴 …覺得也蠻開心的。而回想自己過去，對小朋友好像沒什麼耐心，也從來不敢想像／無法想像自己當爸爸的樣子。基於種種的考量 … 大概在 36 歲之前的我們，心裡想：也許就這樣沒孩子也不錯 … 。當然，那是那時候的想法。

隨著年紀變化的心境

其實我們很清楚，36 歲就算是「高齡」了，特別是對女生來說。如果這個時間不做個決定，大概就會確定是2個人的生活了。

但是，有時遇到以前的好朋友。看著他們從單身、到結婚、到有孩子，到孩子會叫我「Uncle」，那種感覺 …. 很難描述。心裡有時也會想「如果我們有自己的孩子，會是怎麼樣的？」。有時也會想著「如果有一天我們變老，會不會因為沒有孩子的決定 … 而覺得後悔？」。別誤會，我從來不覺得會有「養兒防老」的想法！孩子是獨立的，會有他自己的人生。我只是會想「如果我們決定沒有孩子，會不會未來為這個決定後悔？」。當然，人生沒法重來，我只能想像 … 卻不知道答案。還有，我們年紀越來越大，也不曉得真的想要 … 會不會還能有小孩？

大概到了 38 歲時，我經歷了許多生命的轉折。包含媽媽罹癌、事業經歷過最低潮、甚至跟老婆都短暫分開後復合，到了 39 歲，似乎我渡過了最低的那一段，一切開始有些起色。也許是經歷過這些生命的歷練跟磨練。心態上也會有更大的轉變 … 我們開始會想：「現在的我們，如果想要有自己的小孩 ….還來的及嗎？」

上天給的禮物

老天保佑，我們在 42 歲擁有了大女兒云寶、在44歲擁有了2女兒安寶。因為老婆跟我同年，所以算是醫學上的「高齡產婦」了（當然外表看起來還是很年輕啦！）。當然，過了 40 歲後才當爸爸媽媽，對體力上還是會有一些挑戰的，特別是對媽媽而言，經歷過生產、哺乳、照顧孩子、還有許多的事情，真的是有挑戰的（謝謝老婆！）。所以，我也盡可能接手我可以處理的角色，從一開始的嬰兒用品採買（詳見我寫的超級奶爸系列文）、到訓練寶貝睡過夜、到副食品製作，到大一點後上下學的接送，以及平常有空下廚煮菜給孩子們吃、甚至最近開始帶著他們出門工作。

這麼說好了，我一路在學習怎麼當一個更好的爸爸，也努力做好「爸爸」的角色。孩子們在家裡依賴媽媽（媽媽比較多關愛），也喜歡跟爸爸在一起（爸爸比較不囉嗦）。孩子們也會跟我撒嬌、也喜歡跟我膩在一起。我們會一起看書、逛書局，一起運動、打羽毛球或游泳，他們練武術時我陪伴，他們練琴時我練薩克斯風，我也教他們怎麼用便利貼寫作文、用心智圖規劃一天行程、用蕃茄鐘做作業衝刺。看著她們逐漸長大，長成屬於自己的樣子，某些部份像我們（我跟太太），但更多的部份像自己。我總是覺得很滿足、很幸福！而且，我覺得因為有他們，我慢慢的成為一個更好的人

更柔軟更有愛

不曉得是不是工程師個性使然，雖然我自覺是很認真的人，但有的時候太認真 … 反而變的有點嚴肅，或是太重視「事情」、反而忽略了「感情」。但是孩子的出生，讓我覺得自己變的更柔軟、更有愛。例如：有一陣子孩子不小心尿床，我們第一個反應一定是處理感情：「沒關係，寶貝你還好嗎？」，然後才是去處理事情：「我們還在練習 … 晚上睡覺前記得上廁所哦 …」。面對「孩子」，慢慢的我有一些改變 … 。

倒不是我有耐心，而是帶孩子的歷程，讓我訓練的更有耐心。因為從孩子的角度，一切本來就是新奇，所有的事情都是學習。白話的說，孩子本來不懂，有問題是正常、搞問題是日常。有了孩子後，會「強迫」讓爸爸媽媽更有耐心（懂的都懂！）。從嬰兒時期的哭哭鬧鬧、把屎把尿，到小小孩時期的探索為什麼。到慢慢長大開始上幼稚園小學，不同的時期會有不同的挑戰。當然，你可能會想 … 生個孩子來折磨自己，不是找罪受嗎？哈哈，這個每個人有不同的詮釋，我無法以偏概全。有時候真的有點辛苦，甚至也會犧牲一些自由（一個人最自由是吧，哈）。

但，至少我的兩個女兒，帶給我的是很大的滿足，以及愛！像從小我從不吝嗇的常跟孩子說：「爸爸永遠愛你們！」。隨著孩子慢慢長大，有時會她們也會問我：「爸爸，你會永遠愛我們嗎？」。我總是毫不遲疑的回答：「非常愛，每一分每一秒都愛！」。這裡說的「愛」，不是掛在口中的，而是從心底深處發出的愛！這種感覺只能意會卻無法言傳，但相信有孩子的爸爸媽媽都能感受。我覺得因為有了孩子，讓我更能體會到無條件「愛」一個人的感受。（爸爸，可以陪我上廁所嗎？然後就看到我拿著椅子坐在廁所門口，幫孩子看著門 … 哈！）

更完整的生命體驗

寫到這裡，我其實想到身邊有許多朋友，可能因為種種的原因而沒有孩子，我也要表達尊重、理解、與體諒。每個人都有自己的故事，只要自己的選擇，就是最好的選擇。所以我只是分享我自己的體驗，並無意、也無權影響任何人的人生。

有了孩子之後的人生，讓我有了更完整的生命體驗。因為真的是當了「爸爸」，才會學習怎麼當個「爸爸」。而爸爸這個角色，跟我以往的生命經驗都截然不同。

我記得我第一次抱到寶貝們的感覺、記得用背巾帶著她像袋鼠爸爸的感覺、記得推著車帶她們散步、也記得從單人推車到雙人推車、也曾騎車載著她們，還有早上起床後背著她們去廁所洗臉刷牙，我當然也記得送她們上幼稚園、上小學、練武術的種種記憶。我享受著她們對爸爸的信任及愛、還有愛吃爸爸煮的菜。許多一起旅行、一起露營的回憶，也許她們長大後會逐漸忘記，但我卻會一輩子記得。

這些美好的體驗，絕對是我當了爸爸之後才擁有的。我常常感謝孩子們，讓我的生命更完整。當然我也相信，隨著孩子成長，之後一定還會有許多不同的挑戰。但我們邊學邊過，一定會平順渡過的。

更懂得感恩、也回報父母養育之恩

真的是有了孩子之後，才真的能體會到父母也是這樣拉拔自己長大的。「養兒方知父母恩」，其實有時我們對孩子做的事，真的只有身為父母才做的到！幫孩子換尿布、洗屁屁，都是最基本的。餵食、吃飯、照顧、關愛 …. 。從小到大，太多太多事情了。如果真的用很「理性人」的角度思考，我真的不認為孩子這一輩子，有辦法回報我們照顧養育的恩情 ….。

同樣的，我們也是！當了爸爸媽媽後，我也不認為我們這一輩子，有辦法回報父母照顧我們的恩情！因為，從我們出生那麼小、那麼無助，到今天長大，擁有自己的小孩。我們的爸爸媽媽對我們養育的恩情，我們怎麼可能報達的了！

有一天，我突然想通這件事！這其實是個傳承！我們的阿公阿媽養育我們父母，父母得以成長茁壯養育我們，而我們長大後再養育子女。阿公阿媽的養育之恩傳給父母，父母的養育之恩傳給我們，而我們帶著這樣恩情養育之子女。雖然我們無法完全回報我們父母，但是透過傳承，這樣的愛會繼續傳下去 。

有了這個想法後，我就會想著：「我教養孩子，其實也是對父母之恩的回報」。而我也不求孩子的回報，因為將來她們也會把我對她們的愛，繼續傳承給下一代啊。

小結

因為某次車上跟好朋友的對話，促成我寫了這篇文章。也重新面對了自己從不想生孩子、不敢生孩子、到想要有孩子。最後在上天的幫助及老婆的辛苦下，擁有了2個寶貝女兒。當然孩子還小（一個11歲、一個9歲），但卻帶給我許多成長、快樂，滿足、以及許多感動的時刻。因為有孩子，完整豐富了我的人生體驗，並且讓我有一些機會，回報父母養育我們的恩情。我教會孩子們很多東西，孩子們也給我許多體會。最終這樣的經驗，讓我有機會成為一個更好的人。

這2個禮拜，我分別帶著姐姐跟妹妹，跟我一起去澎湖／花蓮演講。如果是我隻身一人，我總是快去快回，行動迅速。但帶著孩子，我就選擇多留一點時間，慢慢來。雖然一切看起來是慢了一點，但跟孩子一起出門工作，樂趣跟滿足感卻多了好幾倍。孩子跟我說：「爸爸，有人知道我很幸福嗎？」。我回「幸福不用比較，但我們感恩，也謝謝老天給我們幸福！」。但沒有說出口的是：謝謝孩子們，給我這麼多的幸福啊！
