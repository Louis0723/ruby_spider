最近好朋友 Jerry 出書，特別邀了他來家裡聊聊坐坐。問了一下近況，除了出書忙碌外，佔最多時間的卻是前一陣子出的小意外！因為開車被追撞，對方沒保險，自己的保險又只有強制險，結果為了車子修理的賠償跟對方談不攏，搞到需要談判調解、甚至出庭上法院，他覺得真的快煩死了。

聽完他的敍述，我請他喝杯茶降降火，然後問了他一個問題：「如果有2份工作，工作內容一樣，辦公室地點差不多，連同事關係及未來發展也接近相同。就只是薪水不一樣，一個 33,000，一個 30,000，如果有一個年輕人要找工作，你會建議他選哪一個？」。Jerry 聽了後，回答我：「如果工作內容都一樣，客觀環境也差不多，那 …當然選錢多的那一個！」

我聽了之後笑一下，再接著問：「那如果錢少的那個工作，因為很欣賞這個年輕人，因此開了個條件：在薪水不調整的狀況下 ，願意提高福利！因為年輕人每一天開車上班，只要來我們公司工作的每一天，接下來所有開車的意外損失，就由公司扛！撞到別人，公司賠！被人撞到，公司也負責出錢修好。甚至在修車的日子裡，公司還給你一台車子代用，讓你方便上下班 … 」

Jerry 聽了之後笑著說：「哪有這麼好的公司，我聽都沒聽過 … 這不可能啦！有這種公司，不要說一個月差 3000，一個月差個5000～6000，也應該趕快跳槽去啊！」

我看著 Jerry ，跟他說：「有啊！你現在的工作也有這樣的福利啊！」Jerry 笑著說：「最好是有啦！福哥你不要開玩笑！」。我再次跟 Jerry 強調說：「有的！你現在的工作也有這樣的福利！只要你一個月拿3000 出來買保險（這個案例裡指的是車險！）！你就可以擁有一樣的公司、一樣的保險！」。沒錯啊！工作內容都一樣，只是一個月少賺 3000，卻多了這堆保障！

 

注意力才是最寶貴的資源

前一陣子，我剛好也遇到了追撞事故。那天載著孩子，從台灣大道要回家，就當我停在離家不遠的紅綠燈路口前，在車子完全靜止5秒後 … 突然後車撞上來！讓我跟孩子們嚇了一大跳！

當然，人沒事就好！我理智的報案、做完筆錄、安排老婆來接走孩子（流程有寫在 FB 上）。然後，車子進廠。在靜止狀態，被時速 50 km 的後車撞上（事後判定我零肇責），車子受傷不輕！原廠估價金額超過 30 萬以上！更誇張的是：肇事計程車司機，雖然有保險 ….但卻一直不去辦出險！我一直等著對方保險公司出來處理，但卻沒有人理 … 。如果就這樣一直等下去，也不曉得要等到哪一天，少了一台車，上班、接送小孩 … 什麼都不方便。

 

至少有保險

等了幾天，我的保險公司就直接安排車子進廠維修（乙式車體險），反正到時就由保險公司代位求償（白話說：就是不關我的事了！），然後修車的日子，保險公司再給我代步車（代步車險），那一個月 … 我的生活並沒有太大的影響。雖然還是有一點點小小心煩，但至少注意力沒有太大的浪費！

被撞的那一天是 6/14，車子是 6/19入廠，而 6/24 就是《教學的技術》募資開始。修車＆代步車的那個月，就是我創下募資記錄的那個月！很難想像：如果我沒有買好保險、買夠保險，那一個月我的注意心跟心情，會有多大的損失！單單是煩心什麼賠什麼不賠，然後對方要不要出來處理 … 我一定會心情超差的，更不用說什麼創記錄了 … 。

而且，這還是我「被撞」，如果事情反過來 … 你才真的知道保險會有多麼重要！（加買第三人責任險、高額的超額險 …等）。所以，現在只要車子沒保險，或是沒保夠，我是不開的！像那時保險公司提供給我代步車，我也一定要找到：有提供100％ 完全免責保險的租車公司，我才要開。（目前和運＆格上有，1天 300 元…）

 

車子重要，那人呢

如果車子的保險都這麼重要，那人呢？

老婆生老二時，在月子中心突然發高燒，我緊急送她去醫院，那時她剛生產完需要休息，單人房是最好的選擇。但你也知道 … 單人房要自費啊！（1天 6000！）。但還好有保險，讓我奔波在月子中心（安安剛出生還在那裡）、醫院（老婆住院中）、保姆家（云云白天保姆帶），至少不會再煩心住院所支出的花費。而先前媽媽住院，保險也在過程中幫了一些忙，至少看護的錢都有了 …。

老婆身上的保險，其實保護的人是我。換一個角度，我的保險，其實保護的是 … 我愛的家人！

因此，有了孩子後，我加買了更多保險 XD。買了新房子 …. 為了房貸，我再加買了定期險。更不要說醫療、意外、以及許多有的沒有的保險了！

別擔心我亂買，忘了福哥之前做過保險業務員嗎? ^^。只要抓住一個大原則，讓儲蓄歸儲蓄，保險歸保險。大方向就不會錯了！

當然，買好保險，買對保險 … 也是一門專業。也許你可以找一下您信任的業務同仁，他／她會依照你的需求，給您最適合的建議。

但如果可以，真的建議大家花一點時間，研究一下保險，不管是人身保險、還是財產保險（例如車險，甚至很多人忽略的機車保險）。你花的這些時間，最終保護的 … 不止是金錢財產，更重要的是：你的注意力。

時間，才是我們最寶貴的資源！ ＃其實保險也可以改變命運但今天寫不完了哈
