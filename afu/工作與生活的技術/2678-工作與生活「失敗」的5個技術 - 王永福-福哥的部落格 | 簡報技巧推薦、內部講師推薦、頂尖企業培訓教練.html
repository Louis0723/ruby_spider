寫了這麼多工作與生活的技術，並不是只想給大家一些「雞湯文」！而是把過去在我身上，真實有用的方法及技巧跟大家分享。每一個方法或技術，一定是我自己長期使用覺得有效，並且多次改進之後，才整理成文章給大家。而為了更確認效果，我在自己身上進行了一整年的實驗，同時執行多個專案：博士論文、教學技術線上課程、寫書，還有兼顧健康、運動、以及家人。一年過去，看起來都高標達成，看起來真的是有用的！只希望用這些技術，能給大家一些幫助及影響，讓你成為一個更好的自己！

前幾天重看了查理.蒙格（股神巴菲特的合夥人＆最佳戰友）的書，再次複習了蒙格把事情倒過來想的智慧！也就是先想想看「絕對會失敗」的方法，然後想辦法避開它，就更有機會邁向成功！套用這個模式，我們也可以先反過來想：「工作與生活更糟的技術」，思考一下怎麼樣可以讓工作與生活變的更糟，先來點「負能量」（受到每天來點負能量－育聖老師的啟發），也許才可以平衡太多的正能量。

好吧，接下來就是努力思考，倒過來想，給大家「工作與生活失敗的5個技術」

 

1.不要學習

絕對不要學習，學習是那些弱者做的事情！天縱英才如你，不需要任何的學習！如果你是名校畢業，那太好了！你可以憑著學歷打混一輩子了！（名校吔！）。如果你不是名校畢業，那更好了！反正學習本來就跟你沒有關係，你更可以心安理得的打混一輩子！

一本書那麼貴，省下來可以多吃一餐美食！如果一本書的錢不夠，那多省幾本！然後，不要去學習任何新的東西，那太傷害腦細胞了！不要再回學校上課，好不容易才脫離學校，笨蛋才要再回去！不要在工作中有任何新的學習 ，反正你拿的是白金飯碗，世界不會變的太快的！一定會等你的！也絕對不要從網路上免費學習任何新知識，那太浪費時間了！記得，不要學習！把任何學習的時間，都用來浪費與無所事事，這樣過幾年下來，你一定會越來越習慣空洞的生活，也可以確保你工作與生活越來越空洞！這樣朝向我們「失敗」的目標，就不遠了！

 

2. 不要堅持

任何事情，永遠只要有3分鐘熱度，絕不堅持任何事情，反正只要試一次沒有成功，就應該放棄！堅持太花精神、腦力與體力，笨蛋才會去堅持！像你這麼聰明的人，應該做更聰明的事！

早起失敗一次，那就應該放棄！床那麼溫暖，白痴才早起好嗎？運動那麼辛苦，又流汗又喘，幹嗎要運動啊？反正每個人都會老會死，多幾年躺在病床不是很好嗎？另外，設定什麼目標是最沒用的事，設定目標又不一定會達成，無為而治順其自然不是很好嗎？幹嗎堅持變成一個更好的自己？最好是連試都不用試，什麼都不要試，遇到困難就逃跑，不要堅持！一次都不要堅持！相信我，只要什麼都不堅持，「失敗」一定很快會朝我們接近，那就太棒了！

 

3.亂交朋友

要努力挑一些壞朋友，最好是吸毒酗酒樣樣來！你要相信你自己自制力，反正壞朋友的壞行為，是他們自己的事，你一定可以清者自清、濁者自濁！讓自己每天跟壞朋友接觸，每天處在許多「酒色財氣」誘惑的壞環境中！相信自己，只要時間夠久 … 自己也會習慣這些壞行為，然後就會同流合污，一些都會變的更自然的！

記得遠離那些有好行為、好習慣、甚至是鼓勵你、規勸你的朋友。跟這些人在一起太累了！每天看著他們進步、成長、努力，真是太煩了！生活有必要這麼辛苦嗎？還是每天酗酒，甚至用一些違禁品，還是鬼混七逃 … 來的爽快一些！只要身邊的朋友夠糟，並且排除所有行為好的朋友，這樣一定可以有一些抵擋不住誘惑，讓自己掉入混亂生活的深淵。反正久了，也就習慣了！

 

4. 酸民心態

一定要學習網路世界最新的酸民心態！看到很棒的人、很棒的事，一定要用以下三個萬用句：

「啊不就好棒棒」
「這不可能啦一定是編的」
「這很爛沒什麼了不起我只是不想做而己」

只要用這三個萬用句，你就可以否定所有的一切，用你的「酸」來融化所有值得學習、值得參考的訊息。反正遇到一切訊息，記得先酸先攻擊！酸的越嚴重越厲害，大家才會記得你！為酸而酸，讓每一個有用的訊息都被你融化，這樣才能顯現出你的獨特與人格！反正隱藏在電腦後面，酸人也不會被發現，你說是嗎？（可是這樣就不能被記得了 … 還是用實名制來酸好了 XDD）

 

5.不要懷疑

記得相信所有的一切，不要懷疑！不要懷疑他們：包含你的老師、朋友、長輩、甚至父母說的一切！什麼都相信！把未來的一切交給別人！他們叫你做什麼就什麼，讓別人決定你往後的人生！這樣最輕鬆！反正「他們都是為你好！」，未來真的搞砸了，他們也「一定會」為你的人生負責！相信他們過去20年／30年／50年的智慧，用他們的過去，來決定你的未來！這實在是最輕鬆最棒的選擇！

不要懷疑你在書上看到或學習到的一切，要相信每一本書上告訴你的知識（如果兩本書的觀點剛好不同，就選一本、然後丟掉另一本 XD）。包含我寫的跟我說的事！只要你什麼不懷疑，把任何事的決定權交給別人，讓腦子停機不思考 … 那一定可以自己一步一步走向失敗！反正，這一切都是 they 的錯！

 

小結

其實把事情反者想，有時真的是一件有趣的訓練。先不要想怎麼能用「工作與生活的技術」，讓自己成為一個更好的自己，擁有更好的人生，實現更美好的目標。而是倒過來想，要有什麼樣「工作與生活失敗的技術」，才能讓自己擁有混亂的人生。如同我們提到的：不要學習、不要堅持、亂交朋友、酸民心態，以及不要懷疑。只要一直做這幾件事，最好同時進行這5大項目，那麼不管是工作或生活 … 一定可以朝向越來越混亂邁進。

至於這是不是你真的想要的？沒關係 … 相信自己，久了 .. 就會習慣了！
