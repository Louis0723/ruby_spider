熟識的朋友問：「每天都早起？！那意志力一定要很強吧？」。我聽了總是笑一笑，「意志力是很寶貴的資源 … 每天都用意志力早起 … 那也太痛苦了！」。朋友好奇追問：「不用意志力？那該怎麼做？」，我回「用習慣啊！讓一切規律化、自動化 … 習慣了之後，就不需要用到意志力了！」

最近剛好有一本書「習慣力」，書中的內容証實了我的看法。書裡談到生活中有一半的動作，其實是我們無需思考，就會自動執行的。像是走路、開車、洗澡、穿衣服 …等。甚至連娛樂也經常變成習慣的一部份，像是習慣滑手機、習慣看電視 ….，這些都是習慣！意思是不用思考、自動會去執行的！我想，你應該不用動用「意志力」，才能開車，或看電視吧？哈！
把有效工作變成生活習慣

但，你有沒有想過，如果把有效工作的生活方式，也變成是一種習慣 ? 那會是怎麼樣？

這其實就是我這1～2年所做的親身實驗，因為想在50歲這一年完成自己的三大天命計劃：博士畢業、教學的技術線上課程上線、工作與生活的技術出書 … 。所以我必需更有效的安排工作！而早起是其中一個重要關鍵（這個之前文章中寫過了）
但其實每天5點前起床，並不太需要動用到意志力，甚至連鬧鈴都不用！因為當一切變成習慣，早起工作就像開車或走路一樣，自動會發生！我總是像個機械人一樣，在大約 5:00 起床，然後用拍照FB打卡記錄起床時間（記得馬上關機，不要滑手機 XD），接下來走到 La Pavoni 咖機旁加水開機，之後走進浴室沖澡喚醒自己，然後出來時咖啡機也預熱好了，沖一杯 3x Espresso 一乾而盡！然後坐到書桌前，拿起便利貼寫上今天的工作重點，接下來按下 25min 計時器，開始一早的工作。
重點是動作刻意相同，每天如此 … 我甚至可以從起床＞開始工作的時間差，知道今天是那一個動機稍微拖到了一點時間，是沖澡比較久？還是泡咖啡花了一些時間。

刻意培養習慣的方法

而這樣的刻意，就是利用習慣培養的法則，讓自己自動進入到有生產力的工作模式！這樣就無需每天動用到意志力 … 也就是無需痛苦，把意志力用在更關鍵的地方。方法拆解如下：

1.重覆：習慣培養的第一要件就是重覆！重覆的次數夠多，就容易變成習慣，不加思索就可以執行。有人說重覆 21 次、也有研究說重覆2個月～3個月。反正就像刻痕一樣，重覆多、印記深！

2.一致：一致性是習慣培養的第二要件，因為每次的流程都一樣，所以在第一個動作後、就會接續第二動作、然後第三動作 … 只要開始啟動，之後就會自動開始。

3.獎勵：習慣要能強化，獎勵也是過程中很重要的關鍵！像我每天早上 FB 打卡記錄起床時間跟大家問安，看到大家的留言，這其實就是一種外部激勵！而每天完成一些工作，劃掉便利貼上的重要工作項目，慢慢的看到工作有所進展，在一陣子之後會蠻有成就感的，這是更大的內部激勵。

4.移除干擾：在習慣建立的過程中，有些干擾會破壞習慣的建立。例如我一早起來開始工作時，一定會把 FB ／Line ／Email … 全部關掉＆設靜音。甚至有時在重要工作時，會把手機丟在別的房間，這都是為了讓我不會受到這些干擾的影響。
用意志力建立習慣、再由習慣接手建立自己

除了早起工作外，包含像運動（固定每週運動、重覆＆一致，並用三鐵競賽＆運動記錄來創造激勵）、寫作（上高鐵後打開電腦、開始打字、51分鐘後逼自己產出，然後丟到 FB 公開創造激勵），或甚至是自我要求（課後一定 AAR … 這算是好習慣嗎? XD）。生活中的很多動作，我已經不需要考慮太多，該工作就工作、該運動就運動、該寫作就寫作、該「請重作就請重作！」（給別人＆自己課後回饋 XD）。這些都跟意志力無關，讓自己進入自動化的習慣模式。

當然，在培養習慣的初期，還是需要一些意志力的，就像發射衛星上太空，前進還是要用意志力當火箭，讓自己脫離原有慣性的引力。但是，要把意志力用在對的方向：也就是用重覆、一致、獎勵，讓自己更快建立習慣。過程中把干擾盡量移除。等到意志力把自己的習慣推到軌道上 … 一切就可以自動運行，不需要費力了！

你有什麼想要建立的習慣呢？設計一下，如何利用重覆、一致、獎勵，讓自己變的更好吧！也歡迎你跟我分享心得哦！
