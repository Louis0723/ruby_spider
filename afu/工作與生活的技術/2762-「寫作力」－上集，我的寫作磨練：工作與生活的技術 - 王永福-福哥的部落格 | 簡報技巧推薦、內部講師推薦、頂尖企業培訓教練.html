如果有什麼能力，是我覺得最好能具備，並且好好修練的。就是寫作力、銷售力，與網路力。

能用文字自在的跟別人溝通，並且完成期望的目標，這是我對寫作力最基本的定義。沒錯，你看到「跟別人溝通」這個關鍵字了。因為如果只是自言自語，那怎麼寫都好。但如果目標是跟別人溝通，達成期望的目標，那就真的要有一些文字的能力了。

別急，我當然理解，有些書寫是寫給自己看的，只是要表達心情、抒發想法，這樣當然很好。但如果您多了一層目的，在你用文字抒發想法的過程中，別人看了之後，也能懂、也能體會、甚至有同樣的感受，那才是我說的「寫作力」。溝通只是最基本的要求，進一步的要求，是透過文字「影響別人」，這就需要更高階的寫作力了。

以我寫的書為例（上台的技術、教學的技術），讓人看的懂，只是最基本的要求；進一步的我們還希望讀者能受影響，這才是我們的期待。

甚至更小的目標：我們當初在線上課程的募資 Po 文，讓人看的懂，也是最基本要求；而讓人受影，才是我們更進一步的期待。

從讓人看的懂、到讓人受影響，這絕對是不同層次的寫作能力。

我的寫作之路

學校時的記憶，已經離我太遙遠。但沒記錯的話，我高中聯考作文應該分數都很高。只是後來沒有讀高中，而是選擇讀五專土木工程科，接下來都只有鋼筋與混凝土，還有我愛玩的電腦。寫作似乎跟我的工作都沾不上邊。

一直到 2006 年，我正式創業，轉職成為講師，我才開始架一個 Blog，取名為「福哥的部落格」 ，並寫下第一篇 Blog 文章。（2019 年更新為 WordPress 版本：新版福哥部落格）

一開始只是因為對電腦熟悉、然後剛創業也沒什麼案子，心想寫 Blog 也許也是讓自己品牌曝光的一個方法，於是開始寫下第一篇 Blog。一開始也沒講究什麼寫作方法或技巧，就只是講一個「獅子與兔子」的故事，然後再收一個生活上的啟示，有點感覺像小故事大啟示這樣（好爛XD）。反正就從那個時候，開始寫 Blog。

結果，一整年才寫了3篇：除了那篇「獅子與兔子」、接下來寫了「教與學」（想不到我在當講師的初期，就在這個問題了啊！） 。第三篇寫「報告與念稿」，記錄了我在大學兼課時看到的狀況（簡報技巧教學的前身）。現在回頭看到這些記錄 … 好有意思！

 

電腦教學

2007 年，因為公司網站架設關係，我開始接觸 Joomla 系統，那時邊玩邊把對這個系統的學習心得，寫成一個教學網站，Joomla 123。老婆說：「你寫那麼多電腦教學，為什麼不整理成書啊？」。開啟了我出書的第一個念頭。後來找到了碁峰出版社願意出，我用了 42天，寫完了人生第一本書，8萬個字、600 個操作步驟截圖的腦教學書。(免費下載）

第一本書賣的還可以，以電腦書而言，也登上分類排行冠軍。很多讀者回饋我說：書寫的清楚、步驟明白，很容易看的懂。這沒有什麼秘訣，方法就是：我一開始是請好兄弟坤哥來家裡，把 Joomla 的操作步驟一步一步教他，並且把教學的過程錄下來。之後在書寫時，就按這個結構及方法寫下來。有時忘記了，就再回頭放一下當初的教學錄影。就這樣因為邊看邊寫、邊寫邊看。所以我文字的流暢度還不錯。至少一定很口語 … 因為當初就是先口語，才有文字的啊！

 

工作需求

從2008開始，也許是因為創業生存壓力，那年寫了84篇！意思是：扣掉一些年假及大假日，看起來平均每週會寫接近2篇。那年的7月寫22篇（每個工作日都寫一篇）、8月寫17篇（每週四篇），開始累積文字量。看了一下主題：寫的最多的是「簡報」，因為那時開始全職成為簡報講師了 … 雖然很不紅，但沒有案子的我，更應該有產出。所以包含簡報學習心得、自己對簡報教學的想法、甚至翻譯國外的簡報文章（簡報禪在還沒出書前，最早中文授權文章就是我翻的）、還有讀書心得、企管知識、生活感想 … 反正想到什麼就寫什麼。每篇大概都 800 ~ 900 字。以一個禮拜 2 篇的速度，持續增加。

因為對電腦相對熟悉（我玩電腦好久了啊 …），寫作還有另一個目的：就是增加 SEO 的自然排名狀況，讓 Google 上可以更容易用關鍵字找到我寫的文章。因為持續的寫 … 開始在「簡報技巧」領域，我在關鍵字的排名逐漸上昇。那時還幫一間寵物洗毛精製作網站，開始要寫更有行銷導向目的文章

這陣子的寫作，比較像是帶有行銷目的。除了記錄想法外，也希望大家可以理解，並且受到我文章的影響，可以進一步帶來後續行動。除了慢慢累積一些文字量外，基本上，好像也沒什麼寫作技巧。但回看這些文章，開始會看到一些簡單的 SOP：開頭、例子或故事、學到的啟示、收尾。就這樣。

正式出版

就這樣寫著寫著，終於有一天 … 被貴人看到了！

2013/3/28，我的 FB 訊息突然跳出「博士是什麼？的文章，是您翻譯的嗎？好極了，我是何飛鵬，有幸為友。」。沒看錯吧 … 商周創辦人，自慢系列的作者，也是許多人的職場偶像：何飛鵬社長竟然私訊給我！原來是社長看到有人轉傳我 Blog 上翻譯的一篇文章「圖解－什麼是博士／The illustrated guide to a Ph.D.」，又覺得我有尊重原作並取得作者授權，因此寫了封訊息鼓勵我。就從這封訊息開始，開啟了我商業書的寫作歷程。

第一本「上台的技術」出版於 2014/12/25，也就是從跟社長見面到書籍出版，大約花了1年半的時間。本來以為把教課的東西寫成書，應該不算太難 …. 。結果真是大錯！拖稿拖了快半年，什麼都沒寫出來！後來一發狠，決定開始嘗試每日寫作計劃！也就是1天寫一篇，連續30天！（假日有休息 XD）。記得那時二女兒安寶剛出生，在月子中心也還在繼續寫作 … 就這樣一直寫寫寫，先有一個大概的架構，再一篇一篇累積，一篇一篇交給編輯，最後集結後再大修一下（修的更順一點），最後出版成書，蒙大家支持，第一本書銷量還算不錯一也因此上了商業周刊的報導。

第二本書跟憲哥合寫，也是非常有趣的經驗，兩人文筆風格不同、寫作方式不同，卻彼此合作同一本書。寫作的時候彼此觀摩，相互學習，他寫一篇、我跟一篇，他再寫一篇，我再跟一篇。就這樣兩個人累積了50幾篇，然後每一篇文章，彼此再相互寫下一些評語跟註腳。最終出版了「千萬講師的50堂說話課」，有了第一本書的寫作經驗，這本書的出版過程算是平順愉快。再怎麼說：也只要寫一半就好啊！哈！

第三本書《教學的技術》，那時已經開始在寫博士論文了。我的論文主題算是「教學的技術」學術版，寫到一半有點卡住，再加上 MJ 啟發的一些點子，我決定先心裡對教學的想法，先寫成書！寫論文卡住的痛苦，轉化成我寫書的動力。印象深刻的是那時一家人去了Bali 島，在充滿南洋風味的海灘上，每天固定寫一篇。這時寫作對我而言已經像是一個讓自己放鬆的休閒，想到什麼就寫什麼 … 有時常常在動筆之後，才發現原來這些文字，是隱藏在心中的想法。但在寫下來之前，自己也沒想過可以寫這些！就這樣寫寫寫 … 又集結了 16萬字，出版了很有厚度的《教學的技術》，除了出版時登上多週的排行版冠軍（還有大年初一、初二、初三的冠軍 ！哈！），最後還成為 2019 年博客來分類排名前50大暢銷書 。也算是有不錯的表現！

現在在寫的第4本「工作與生活的技術」，在經歷了這些年這幾本書的訓練，還有論文寫作的磨練。寫作已經算是除了說話外的另一個能力，心裡想寫什麼 … 大概就能寫什麼。想好主題後，每天都寫個 1000 ～ 2000 字也沒什麼問題。前2週還試著連寫了2個禮拜，大概也有 3～4 萬字的產出。現在唯一欠缺的，就是固定寫作的時間（還有很多其它的事要做），以及寫作的效率了（打字不夠快，1小時大約只能打出 1200 字上下）
