如果沒來現場，你很難想像：120 人的分組討論，是怎麼樣熱烈的狀況！有些人蹲在地上寫壁報紙、有些人站在圍圈討論，隨時老師不斷倒數計時，時間一到，大家𣊬間舉手！踴躍到你都挑不出那一個人手舉比快，因為都太快了！是的，這是一群大家平常覺得過於安靜、過於理性的大人們，大家進到「教學的技術」線上體驗演講的現場，沒多久就變成這樣子了！

這是《教學的技術》線下體驗之夜，前一陣子募資時，我們特別從 5000 位購買課程的夥伴中，抽出 120 位，在今天晚上齊聚城邦書坊的場地，我們讓大家一起看看：花了快一年的製作時間，首次公開播放的《教學的技術》線上課程。就好像電影的試映會這樣。

當然，身為《教學的技術》作者＆線上課程的講者，我不可能只是請大家來這裡，然後大家一起看投影機把線上課程播出來 … 就回家了（這樣的活動也太單調了 XD）。大家好不容易才抽到籤，可以一起齊聚在這裡。我當然要好好設計一下，讓大家滿載而歸 。除了看影片外，我也許可以實際帶一些教學的體驗，讓大家有更真實的感受後，再來看影片的說明，一定更有感覺。另外，我也許可以公開一些，大家不知道 … 但卻是很重要的製作細節，讓大家更了解什麼是「三機三鏡」，以及我們要怎麼把這些不同的鏡位，用同一條故事線串起來。嗯 … 在體驗會開始前一週，我仔細的思考著。然後，在前5天，我用投影片草稿，把流程仔細規劃好，跟我的超級經紀人：牛奶姐過了一次。說到這個，牛奶姐果然內行 … 知道我一定不會只是放影片，而且會帶一些體驗。哈哈，果然有默契啊。

帶體驗、看影片、聊細節，就成為我打算在演講時讓大家經歷的三大關鍵！

種籽發芽公益計劃

19：05, 《教學的技術》線上課程體驗會開始，燈關暗、先放出募資影片 … 很多的記憶，又回到募資的那段時間。真的很謝謝大家對我們團隊的信任，讓我們能創下台灣最大課程平台: Hahow 開站6年以來的記錄。我們在募資之後才對外公開的「種籽發芽－公益計劃」，也把首批免費的線上課程18份，捐給「博幼基金會」，由副執行長親自來現場領取帳號。接下來我們會捐出 500 份帳號，幫助更多偏鄉或資源缺乏的老師們。未來銷售數量增加，我們也會增加捐贈帳號的數量。別誤會，我們是在「募資結束」後，才公佈這個消息 … 表示我們把商業及公益，區隔的很清楚的。但，能夠在行有餘力，做一點小小的公益，我們心裡還是很開心的啊！（未來只要大有知道有需要幫忙的偏鄉學校的老師，或是有需要協助的公益單位，都可以提供給我們做為參考哦 … 持續捐贈中）

帶體驗

公益頒獎結束，正課開始，我照例加足馬力、火力全開。開場建立信任、分組建立團隊、遊戲化建立動機，然後第一個問答法，就引爆現場搶答了！哈！而所有現場的答案，都在我原先的預期之中，接下來的問答法重點影片，相信有回答了大家心裡面對問答法操作的問題。

問答暖身後，小組討論開始！跟過去不同的是：這次我試著在大型現場，引領大家進行小教室才會有的正式版小組討論。6人的分組規模，2用粗筆大紙寫下答案、然後2人小組上台發表 … 。大家馬上進入到超認真討論狀態！有人蹲在地上寫海報、有人圍著圈圈熱烈討論。這種投入的樣子，絕對身為老師最想看到的畫面啊！接下來時間到，討論結束！大家又搶著上台！我只能一直勸告大家「冷靜 … 不要激動 … 控制 … 」，哈！雖然我知道現場會變成這樣，但還是被大家的熱情參與小小嚇了一跳啊！

看影片

只是，這一切也都是規劃好的。在大家體驗完「小組討論法」後，我們首次讓大家看到「教學的技術－線上課程」裡小組討論法的影片。剛才讓大家親自體驗的技術，又一一的再被影片裡的老師（也就是我 XD），精準且完美的示範、解析。感覺就好像剛才在現場的我，又跳進去影片中示範了一次，講的話一模一樣，動作操作一模一樣、學員反應一模一樣；或是更像影片裡的我，剛才跳到現場來做了示範。沒到現場，你真的很難感受那種對比現場跟影片，熱情投入卻又冷靜精準的示範 …. 那種衝擊，相信讓大家的印象是深刻的！我只是想再次証明：教學是一門技術，是一門可以拆解、可以複製、可以模仿的技術。不管是在影片裡操作、或是現場操做，效果都會一模一樣的啊！

聊細節

在感受過後，我進一步揭露了製作這門課背後的細節，讓大家知道什麼叫「三機三鏡」，又要怎麼製作分鏡表，把理想變成實際 … 。還讓大家辨識不同的鏡位，以及從一小段影片中，觀察不同鏡頭的切換。在這每一張實際照片的背後，有太多努力的刻痕！大家才知道，原來「一鏡到底」的背後，是無數的 NG 換來的！（大家應該開心的是，福哥也會 NG …而且 NG 很多，哈）。

從線下到線上

「一定有一個更好的教學方法，只是我還不知道！」，這是我最後跟大家說的話。相信現場 120 人的投入參與，連滑手機、甚至做筆記的時間都沒有。大家應該能體會這句話真正的意義。

不過，現場分享會還是有缺點的！缺點是：因為大家太投入、太踴躍，連滑手機、甚至做筆記的時間都沒有。哈，來現場你就知道！不過開心的是，再過幾天，「教學的技術」線上課程就要上線了（2020/8/31 逐步公開）。您接下來就可以在螢幕前，「控制」福哥的教學節奏，想快就快、想慢就慢，還可以隨時按下「暫停」鍵！叫福哥先不要講話，讓自己先好好放鬆一下，消化完後再開始！哈 … 想到這個畫面，你不是就更開心了啊！（第一次可以把福哥＆教學的技術，掌握在自己手裡 … 的搖控器。）

「多影響一個老師，就能影響更多學生」，希望《教學的技術》線上課程，能給您一些不同的教學觀點及教學技巧，讓您成為更好的老師，影響更多的學生。如果您有任何收獲，也歡迎您跟我分享哦！
