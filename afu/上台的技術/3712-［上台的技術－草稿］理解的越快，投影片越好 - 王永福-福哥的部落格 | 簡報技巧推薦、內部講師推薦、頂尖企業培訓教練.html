這個禮拜打算來寫一下投影片相關主題

當然，這個主題可大可小，可以寫成一整本書（市面上有太多了），但也可以寫成幾個 Tips …

如果要我濃縮成簡單的文字，描述什麼是好的投影片，那可能會是「理解的越快，投影片越好」，或是反個方向「投影片越好，理解的越快」。以讓觀眾理解的時間，當成投影片評估的指標

用這個標準來檢視一下，傳統的一大堆文字 … 理解的應該會很慢吧（連看都看不清），所以字越多 … 字越小 … 應該是越爛的投影片（無誤！）。

那可以用這個標準來 Check 一下其它的投影片作法嗎？我們來試試看：

 

圖像投影片

最近流行圖像化投影片，特別是滿版出血大圖投影片。如果剛好跟內容貼近，讓觀眾能更快理解，當然是好的投影片。如果在圖上再加上一些 keyword （半透明底色＋文字），讓觀眾更能一目了然，理解速度更快，那就更好了（嗯 … 無誤）

但如果投影片只是放了圖像，卻與主題無關，或是圖像本身太隱晦，梗埋太深 … 。雖然有圖像有視覺，但卻不算是夠好的投影片！所以重點在增進理解，而不是投影片圖像啊！

 

大字流（高橋流）

再用這個標準 Check 一下：大字流（高橋流）投影片，重點也不是巨大的文字，而是這些文字要剛好搭上講者所說的內容，幫助台下加深印象及理解。這樣才會是夠好的投影片。但是如果從頭到尾都是大字流（高橋流），觀眾一陣子就會覺得疲勞了，反而到後面無法增進理解，這樣就又不夠好了（過與不及的問題）。所以用理解的角度來檢視，高橋流適合在重點強調，而不是一直大量運用。（嗯 … 無誤）

 

半圖半文字

另外一種是半圖半文字的投影片：一邊放大圖／一邊放簡單的條列式文字。這樣的作法也能幫助觀眾理解。當然，半圖半文字的投影片，圖一定要跟文字是搭配。重點是增進觀眾的理解，而不是圖本身。您還記得吧！

但是當條列式文字過多時，理解的速度又降低了。這樣投影片又變的不夠好了。改進的方法是：讓條列式文字一次出現一段，講到才出現。這樣觀眾一次只集中目光在一行條列式文字，理解的速度又加快了，雖然投影片的形式沒變，但加入簡單出現動畫後，品質就變好了。（嗯 … 檢查無誤！）

 

表格或圖表

再來用同樣的標準檢查一下表格及圖表，傳統的作法 … 一堆密密麻麻的表格，一大堆資料，細小的文字 … 或是一堆擠在一起的柱狀圖。這樣當然不能通過理解速度的檢查。那要怎麼作呢？

簡化及減化 … 是我們可以作的第一件事。拿掉不需要的資料，只留下必要的重點。這當然可以增加觀眾對資料理解的速度。

接下來，加入適當的指標，把我們想強調的重點，直接標示在表格或圖表上，趨勢是上昇的 … 就加入上昇的線或箭號。應該注意或觀察的，就以顏色或線圈起來。這樣觀眾理解的速度是不是更快一些了？

再進一步，我們還可以加入簡單出現的動畫。先隱藏掉還沒講的，畫面只出現我們講到的部份，然後隨著我們說明的順序，一個一個的 Show 出來我們想讓觀眾看的資料。這樣能讓觀眾更快的理解我們想呈現的。(嗯 … Check 無誤）

 

小結

所以整個來看，好的投影片並不是只有一種特別的型式，全圖像也好、大字流也罷、或是半圖半文字也可以，只要能讓觀眾快速理解，那就夠好的投影片。但是如果沒有掌握住「理解」這個關鍵，即使用同樣這些型式製作，也可能是不好的投影片。而表格及圖表也應該掌握住這個關鍵，才可以把投影片作的好。

投影片的方法都是外在的，抓住理解這個核心 … 才是真正的關鍵啊！
