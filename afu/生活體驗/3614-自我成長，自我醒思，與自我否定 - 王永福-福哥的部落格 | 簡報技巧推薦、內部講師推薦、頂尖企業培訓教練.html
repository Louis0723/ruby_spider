朋友上了一門自我成長課，最近看到他 … 反而悶悶不樂。

問他為什麼？「在課程中被教練嚴厲批評，覺得自己要重新醒思自己」

我想到了十多年前，當我還是一個年輕業務時，為了尋求自我突破，也曾經去上過心靈成長課程。

課程細節就不再贅述，大家先取一個外號，然後教練上課，有很多活動，一開始大家都很保守 … 但是教練會嚴厲批評，甚至駡到學員哭。然後會有一些音樂或活動，讓您覺得重生 … 有時會上課到很晚，有時是很嚴格的連坐規定 … 反正搞了不少東西 …在過程中，我雖然有想辦法投入（嗯 … 課程很貴的！！），但是我也會想 … 這樣有用嗎？

不過有趣的是：你很容易看的出來 … 誰剛上過課。譬如：剛上完課的學員，看到人就會說「我愛你」。然後會跟人熱情擁抱 … 。（是的！）有趣的是：上述這個行為，在3個月之後就會消失 … 然後回歸正常 …你問我：「上完課後有什麼改變嗎？業績有變的更好嗎？」我的回答是：沒有！！

但是我的好奇心，促使我去找一些相關書籍，在其中一本書裡 … 找到了這個課程設計的相關資訊。書名好像叫「洗腦手冊！」（正確書名忘了）。但是這是真的 … 裡面詳細的記載：如何抹除身份認同／如何製造壓力／如何創造集體服從效應／如何透過時間、燈光、音樂、活動操作，來讓塑造大家對這個新團體的認同 ….

在上完一期後，我就沒有繼續再上了（很來還跟這個課程單位有些不快，不過也不再細述了）。

我只是覺得：透過課程，有系統化的學習 … 是一件好事！但是如果因為課程，而讓你自我否定，或是讓你更搞混你自己 … 你確定這樣好嗎？

偶有所感，回想過去。簡單的記錄一下自己的想法 … 也跟朋友加油！
