坐在往新竹的高鐵上，我心裡持續準備待會的授課內容。看了一下時間 …. 噫？…. 車子現在應該要減速，準備靠站了 … 為什麼感覺沒有要停止的樣子 …. 嗯？？

下一秒我才回過神來：「糟了！這是直達板橋台北的列車。新竹當然不會停！！」

我馬上 Check 一下板橋回新竹的列車，不幸的是剛好差了1分鐘無法立即銜接上，需要再等半小時才有車。接下來回call 給課程助理，請她跟 HR 回報這個狀況，說明課程會晚半小時才開始。然後再打電話給已經在等我的計程車司機 …

早上要出門前，我心裡就覺得怪怪的 … 好像會發生什麼事？還再次 Check 了一下裝備是否完整 … 沒什麼問題啊？然後出門 … 覺得今天時間好緊，好像快趕不上高鐵了 … 快馬加鞭的趕到月台衝上去，心裡還有一點小小的喜悅。回想起來 …原來從一出門，我心裡想的班次就是錯的啊！

重點是：我幾乎每天都搭高鐵，怎麼可能會搭錯！！（氣！）

搭上回程的車子，9:08 進站新竹，9:10 搭上計程車，接下來司機一路狂奔，9:25分抵達上課現場外，9:26分進到教室，9:29 設備 setup 完成，9:30 開口講第一句話 ….。

「各位好！今天早上發生了一件神奇的事！每天我都坐高鐵來新竹教課….早上，我看到新竹站到了，結果高鐵 … 它沒有停！！眼睜睜以時速300km 刷過去！」。學員們露出好奇的表情。「我才發現 … 我搭錯車了！」學員們這時笑了出來。「不過，一個好的老師，即使不能準時開始 …. 也一定要 … 準時結束，各位說是不是 ^_^」。大家對我報以熱烈的掌聲 …. 我知道場面穩回來了 …. 然後開始一如往常，開始整天的課程 …!

沒有人知道的是：我心裡其實還是又急又氣！很多情緒其實還是留著。但是我知道更重要的是：趕快穩住現場，回復到原本的上課節奏。不要一開口就道歉（因為沒有用，觀眾不會因此原諒您，反而會一開始就覺得有點失望）。反而應該利用這個梗，創造一個有趣的開場 … 至於心裡的情緒，自己應該消化＆平撫，無論如何 … 還是應該要表現好啊！以這個標準來看，我算是作到了！

只是，在中午吃飯時 … 我只吃了二口，整天心裡的情緒，還是緊緊的 … 一直到下課時才放下來，頓時覺得全身酸痛 ….「怎麼會發生這種事呢！真是的！」….（今天真是元氣大傷啊！還好回家老婆陪我去吃美食，然後抱著可愛的女兒 … 才讓自己又平復一點點 … 整個事件….也算難得的經驗，所以記錄一下，也跟大家分享）

 

小結：

1.自己晃神了 … 讓習慣自動運作，問題是今天的自動運作系統搞錯了。結果就就發生坐錯車的事！

2.道歉是簡單的，Hold 住現場才是重要的！

3.問題發生了，要馬上快速的評估接下來的修補方案。但切記不要慌亂，做出錯誤選擇，反而擴大了問題的嚴重性。

4.有問題的一天 … 也許反創造了印象深刻的一天 … 反正問題還是會過去的，只要撐住 … 就會搞定！沒有什麼過不去的啊
