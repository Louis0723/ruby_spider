我們是新手爸媽，大部份朋友總會說：「很辛苦吧？晚上應該常常要起來餵奶吧？」。當我說云云從一個月開始後，就已經可以睡過夜了，晚上都不需要起床餵奶 … 許多人的反應更是驚訝！似乎這是不正常的事！

我們很幸運，云云在經過適當的調整後，很快就可以睡過夜。這其中有些是云云天生個性好（小朋友也有個性上的差別，有的好帶有的不好帶），有些則是我們在過程中所作的努力。有很多朋友知道後，常問我們是怎麼做到的？除了拜讀很多育兒書之外，我們也作過一些不同的調整。因此我簡單的整理一下：在這段日子中，為了讓云云睡好覺，我們所作過的一些努力。當然這只是我們個人的經驗，但也希望以後能讓有需要的父母有一些參考及幫助。

1.多看書抓方向

從百歲醫師系列、超級嬰兒通、每個寶貝都能睡好覺、從零歲開始 …。我們在一開始就看了不少育兒書，先抓到一個大方向：Baby 的作息時間是可以調整的！一覺到天亮是可能的！雖然每本書的說法不一樣（有的說從出生開始調，有的從六個月後開始調，方法有的溫和有的激進），但是這共同的看法都是父母要幫 Baby 建立作息時間表，特別是睡眠時間表。這些書在一開始建立了我們基本的方向及認知，讓我們能有一個參考的作法及依據。

2.建立共識

照顧者本身要先建立共識，是不是彼此都認同這些方法？最怕照顧者（例如爸爸 v.s. 媽媽）方向不一致，到時候一定會引發衝突或相互指責！我跟老婆都清楚讓云云睡的好是非常重要的！不止是為了讓我們晚上無需起床，更重要的寶寶要睡的好，才會長的好，心情也更好！。我們體認到一開始可能會有一段陣痛期 …. 但為了讓寶寶睡的更好，這些調整都是值得的！

有些的家庭除了父母外，可能會有長輩介入問題。無論如何，大家都要能有一致共識 …. 不然任何作法都可能受到干擾，而無法繼續下去（我這麼說大家應該都懂吧 …）

3.讓Baby 睡自己的床

讓寶寶有一張獨立的嬰兒床，除了減少干擾外，也比較安全（防止翻身壓到或跌落）。我們依百歲醫師的作法，用4層浴巾（Costco）＋乳膠床墊，舖了一個安全的嬰兒床。浴巾是為了透氣，減少悶住危險，若有吐奶也可立即吸收，而乳膠床墊是用來取代百歲醫師的彈簧床（是的，百歲醫師在浴巾下還有彈簧床，不是只睡浴巾！）。

但我們把嬰兒床放在自己的床邊，方便我們隨時起身照顧，（百歲醫師是建議獨立一間房間，但我們捨不得）。這幾個月下來，我覺得讓 Baby 有一張獨立的好床是很重要的！像有時老婆親餵，云云會睡在我們大床上，這時我就要超級小心，用一個枕頭隔離我跟云云，免的我翻身不小心壓到她！結果就是 … 我睡的很不好！三不五時也會吵到她 … 所以我覺得還是讓寶貝睡嬰兒床比較好！

4.建立固定上床時間

我們大約抓3~4個小時為一個作息時間，云云一開始大概是 6:00 起床（有時早一點），因此時間就是6:00/9:00/12:00/15:00/18:00/21:00（另外還有一個24:00夜間餵食，詳見Step 7）。也就是大約 21:00 點（晚上9點）上床睡覺。我們會先餵飽云云，抱抱，然後抱她上她的小床睡覺。有時她一面吃一面就想睡了（但我們還是會敲奶瓶叫醒，一直到完全吃飽為止），但常見的是吃飽放到床上還不睡，這時就會進入到下一個步驟，也就是最考驗父母的關鍵步驟。

5.忍受 15 分鐘的哭泣

如果沒有計時，即使是1分鐘，相信爸爸媽媽都會覺得很久！在確定Baby 尿布乾淨、奶奶吃飽、也給他充足溫暖的抱抱後。我依百歲醫師的作法，放她到床上，然後 …. 狠下心！最多讓她哭15分鐘！。我真的是拿著手機計時她的哭泣時間！說真的 … 這實在蠻煎熬的！！後來我參考超級嬰兒通的作法，哭哭時會抱起來惜惜一下，或是小拍拍。但大約30~60秒後，還是「堅定」的放她回小床睡覺 … 再哭哭時再抱起來，惜惜後再放回去 … 反正就是陪著她，但也堅持的讓她在床上睡覺！

這一關很難通過（看著寶貝在您面前哭 15分鐘 ….？！），但卻是重要的關卡。Baby 需要我們的幫助，才能訓練她「自己入睡」，其它任何像搖晃／搖床／推車／甚至奶睡 … 也許短期有效，但長期卻會影響 Baby 的睡眠時間，並且建立寶貝不好的睡眠習慣。但是純綷像百歲的作法真的很不容易作到，也就是讓她自已在房間裡哭到睡著！我一開始曾作過，但我自責到受不了。最後採取超級嬰兒通的修正作法好一些。也就是可以安撫，但一下下後就又要放回去 …。

在上床前記得先檢查Baby 其它狀況ok（尿布／餓／陪伴／沒有生病），然後才是15分鐘計時 ….。如果 15min 後還沒有入睡？…. 我大概會猜她還不想睡，如果抱起她後 … 她還開心的跟我玩，那我可能會跟她玩個15~20 分鐘，然後再重新來個15min。當然爸爸媽媽也可以從她揉眼或打哈欠 … 來判斷寶貝是否愛睏了。關於肢體語言的判斷，超級嬰兒通裡面有更詳細的說法。

6.吸手指也無妨

我們發現寶貝似乎需要吸吮，會覺得比較有安全感。有的父母是讓 Baby 吸奶嘴，而我們則是參考百歲醫生的書，讓云云吸手指。因為當 Baby 很小的時候，吸奶嘴常常會掉，萬一晚上小醒找不到奶嘴就會哭（Baby前幾個月是無法自行抓起奶嘴）。而小嬰兒大約1個月後，就能自己把手指往嘴巴塞，有些醫學書說這樣也可以幫助 Baby 自我探索及認知。重點是云云自己吸手指的動作，也讓她自己覺得更有安全感，隨時想吸隨時有。

有的朋友擔心會不會手指發炎或吸破皮，我們的經驗是不會。云云在睡覺前吸手指，真的睡著後自己會放開（很神奇！）。在5個月之前也經常看到她自己在吸吸，但手指也沒有發紅或有異狀。現在7個月，她喜歡亂咬其它的玩具或布書或小車車，反而吸手指就比較少了。

7.夜間餵食法

Baby 在尚未建立睡過夜習慣前，常常是肚子餓而在半夜起來哭哭。因此在入睡前（我是夜貓子），我大約會在0:00 時再餵食一次，讓她肚子飽飽可以撐過一夜。

夜間餵食並沒有叫醒寶貝，而是在燈光昏暗的狀態下，直接把她抱起，然後快速的把奶瓶塞入她的嘴巴，您會發現 Baby 在眼睛閉著的狀態下，可以把奶奶吃完，吃完後拍打嗝（眼睛還是閉著），拍完後放上床 …. 然後好像什麼沒發生，寶貝繼續睡著！我第一次這麼作的時候，覺得這實在太神奇了！！Baby 真的不會醒，然後吃飽，然後繼續睡 …. 真是超級厲害！但是記得保持燈光昏暗，還有不要跟 Baby玩….這真的有用，也會更快讓 Baby 可以睡過夜。（因為吃飽飽）。詳細的作法請見超級嬰兒通！

8.半夜延遲餵食

我記得最開始 Baby 還沒完全能睡過夜時，在半夜起床哭哭。這時我們會拍拍她，安撫一下 … 然後還是實施 15 分鐘哭泣原則（我知道真的很難受，但這是為了讓 Baby 有更好的睡眠….過程中必要的努力）。如果15分鐘後還哭哭，我們才會餵她。有幾次她哭不到10 分鐘又睡著了（這是我們期待的方向），有幾次哭哭後，餵一下後才睡。重點是：逐漸的拉長半夜餵食時間。其實以我們的作法，0:00 半夜餵食，然後6:00 起床餵（一開始時），中間大約只有6小時，也就是跟白天的4小時餵時週期才差2小時，我們家云云還蠻快就適應了。在出月子中心3~4天後，她就能睡過夜了 … （22:00入睡，0:00 暗夜餵食，6:00 起床吃第一餐）

以上的8件事，大概是我們一開始作的努力。整個過程還算蠻漸進而溫和，沒有太多很「狠」的地方。當然「15分鐘的哭泣時間」實在蠻不容易作到的，但這真的有效！有好幾次云云翻來翻去，甚至大哭 … 但是我忍著放她在床上，不到10分鐘就睡著了。但是當我們不忍心而把她抱來抱去，反而半個小時之後都還沒睡著！經過幾次實驗，我們發現只要寶寶睡的好，心情就會好！重點不在於是否哭15分鐘，而是能否建立好的睡眠習慣。整體評估後，我覺得幫助寶寶睡過夜，真的是很棒的一件事！

另外以下的幾件事，是我們也有注意的一些小提醒：

1.儘量不要奶睡

讓 Baby 一邊吸媽媽的母奶，然後一面自然睡著。這很常見，很溫馨也很有效，但卻可能會讓寶寶半夜醒來！主要是因為寶寶常常還沒有吃飽，就睡著了！所以為了確認她有吃飽（才能撐過夜），晚上最後一餐（以及夜間餵食）我們一般都用母乳瓶餵，而不傾向媽媽親自餵奶。只有少數的時間，當寶寶特別亂（吃飽後，怎麼騙都騙不睡），媽媽才會出動奶睡的方法 … 這算是必殺技，但我們儘可能不亂用以免失去效果或造成依賴。然後在睡著後，我還是會抱她回小床睡覺。

2.儘可能在床上入睡

不論是上述的奶睡、或是搖睡、抱睡、床上哄睡 … 我們儘可能在 Baby 覺得愛睏或是剛入睡時，就抱她起來 … 放回床上。在轉換位置時，有時她會有點醒來或眼睛打開，這沒關係。我們就希望她在入睡時，知道她是在小床上入睡的。這樣就不會造成中間小醒時，發現自己不是在原來的入睡點而大哭。

嬰兒在睡眠狀態每40min~60min 就會小醒來一次，這是本能機制（原始的時候防止睡太熟而發生危險）。您可以想像一下….如果 Baby 都是在媽媽懷中入睡，然後每次醒來都是在小床上（噫？？媽媽怎麼不見了？）…她當然會大哭！而且下一次她可能在媽媽抱時，儘量不要睡著 … 以免媽媽再次消失 … 這些都是錯誤的連結。所以我們儘量建立她固定的睡眠點，只要上床 … 就是要睡覺了。

3.絕對不要趴睡！

雖然我們親自實驗的結果－在 Baby 剛出生還不會翻身時，趴睡真的睡的比較安穩，我們也採取漸進式作法，眼睛都緊盯著觀察，並且依照建議舖透氣毛巾床，自己還用口鼻去實驗不會悶住。

但是，強者我朋友，知名小兒科醫師白袍旅人說：「絕對不要趴睡」！因為美國長期實驗數據，趴睡更容易造成嬰兒的風險！比較起利益，風險成份更大！。特別加此警示！

（這是出生後第二天）

其實等到4個月後云云自己會翻身，這時她自己有時趴睡有時仰睡，一個晚上換了很多種姿勢 … 仰睡趴睡都沒差了。

有研究說趴睡與嬰兒瘁死症有正相關，有研究則說前述研究主因非趴睡，而是厚重綿被及室內高溫 …無論如何，安全才是我們父母最關心的！

剛從月子中心回來時，云云是 21 點睡－0:00 夜餵－6:00起床，慢慢的在3個月大後變成 22:00 睡－6:30 起床（然後吃飽玩玩後，又回去睡）。在過去的7個月，除了前一個月調整期外，其它的時間云云大概有95% 可以一覺到天亮，只有少數幾天因為她感冒生病，鼻塞或咳嗽而在半夜睡不好。

其實比較花時間的，還是一開始的入睡過程，即使云云現在已經7個月大，要入睡的時還是常常需要在小床上滾來滾去爬來爬去10~15 分鐘，好像找不到進入睡眠的入口一樣（但是大部份不會哭哭），這是我或媽媽會陪在小床邊，有時拍拍安慰一下，或唱唱搖欄曲，但還是讓她在小床上入睡。媽媽有時則會抱起她來搖搖，等她快睡著時再放回小床（沒有真正睡著，而是兩眼發直或快眯起來）。反正習慣之後，我們也都知道她會有什麼反應，以及怎麼處理。她也很習慣在那個時間就入睡了，不會有太多的抗拒及哭泣。重點是：一入睡後就睡的很好！

也有幾次的時間，云云真的很累，也許是錯過了一次的小睡（她習慣在19:00~20:00 間又小睡一下），這時反而因為太累，脾氣變的不好而睡不著，抱也不是、哄也不是、玩也不想玩 … 就是哭！我們會再次 Check 一下：尿布 / 餵飽 / 抱抱玩玩，如果不是這些問題，又接近上床時間 … 我就會抱她上床。接一來就會聽到她大哭！！（是的！心情不好的大哭！），我讓她哭個1~2 分鐘，會再抱抱她安撫一下，然後再放回去 … 又大哭，然後再哭個 2~3 分鐘再安撫一下（有時不見得會抱起，而只是拍拍或唱歌陪她）…. 大概在10 ~ 15 分鐘內，寶貝就會睡著了！其實整個過程，我跟 JJ 常常不忍心，想說要抱她或奶睡 … （請注意，前面要入睡時，已經抱她／跟她玩／餵飽她 … 她還是會亂＆哭）。但只要忍一下下 … Baby 也許就入睡了，然後醒來心情又很好了！

為什麼要這麼累？為什麼要這麼狠？為什麼要建立時間表？其實如果我們不幫寶貝建立睡眠時間，常見的經驗是：每2~3 個小時 Baby 會醒來一次，然後哭 … 然後亂，更常見的是小嬰兒一哭，父母的反應就是他餓了，而想要餵他 … 卻沒有想過他可能是累了 … 而應該幫助她入睡（因為塞奶瓶最簡單）。在沒有時間表的狀況下，真的很難判別寶貝的哭聲，到底代表了什麼需求？更難判別，這個時間是不是該讓她入睡了（因為沒有時間表）。這樣的狀況下，不止 Baby 累，爸爸媽媽更是累！然後心情就不好，然後 ….. 我想大家都知道會怎麼發展了。

我們所做這些努力，最後換來的 … 是云云每天起床，天使般的笑容！我覺得這樣的笑容，解釋了所有的一切！也希望以上的心得，能給大家一些參考的方向。願您的寶貝，也能每天睡過夜！

本來云云已經可以睡過夜，後來小生病後，作息又全亂了。生病時總是比較疼，不忍心讓她哭哭，所以那幾天一哭就奶睡，馬上就破功了，一個晚上會起來1~2 次要喝奶。

為了讓云云可以睡的更入眠，還是決定再作一次作息調整。也試了不同的作法：

新的作法，是入睡時，在她背上拍拍，唱催眠曲陪她。拍拍是為了讓她安心。

如果一開始大哭，可以抱起來安撫一下，不哭後就放下。一開始放下會再大哭，再抱起來安撫一下，再放回去，這樣要重覆好幾次。一直到放下去拍拍背，不會大哭（小哭 ok）

重點有三點：

1.修正版哄入睡，拍拍隨著她

2.大哭可小抱，不哭還是要放回去（這一點會有點挑戰）。還是要讓她在床上入眠（而不是您身上）

3.晚上若醒來哭，不馬上抱起，還是拍拍背再哄入睡。

云云現在 1 歲 2 個月，大部份的時候，她都睡的非常好。固定的睡眠流程是：晚上 10:00（有時 9:30）給她睡前最後一瓶奶奶，換一次乾淨的尿布，然後放她在自己的小床上（與我們同一間房，但獨立的嬰兒床），她邊喝奶邊眼睛就會閉上了，然後到隔天約 8 點起床（有時早一點有時晚一點）

有一個最大的改變是：如果她在入睡時哭哭（少於 20% 的機會），我們目前的作法是：在床邊陪她，並且輕拍哄她入睡（但是還是沒有抱起來！）。會輕拍她的背，唱著催眠曲 … 哄著她入睡。一般會在 5 min 內就睡著了。

而半夜有時她會小哭哭（作夢？半夜醒來？或是沒有任何理由），我們還是會輕拍她的背，小小聲哄她再入睡（不會抱起也不會泡奶），一般在 30 sec 內就會停止哭泣，再度入睡！

這個方法是參考超級嬰兒通的作法（網路上還可以看到第二輯－簡體版），與百歲的差別是：比較溫和一點（百歲是放給她哭 … 有時挑戰蠻大的）。我們最終覺得這個方法對我們適用（大約在7~8 個月後開始採用）。也就是：修正了原始哭 15min 的作法，現在只要哭哭，就是在背上輕拍輕哄，有陪她入睡，但是仍然不會抱起，也仍然是讓她在自己的床上入睡。

只有不到 1% 的機會，這樣輕哄也沒用，超過 3~5 分鐘還是大哭（睡前玩太興奮？或是夜驚？）。我們會抱她起來，抱著哄哄 … 看看是不是還要喝奶奶（大部份不要，只是想哭），我們會抱著哄到她平靜不哭，有點想睡（但還沒睡）… 就再放她回去！（有時會再大哭，就再抱回來）。這個放下的作法也是參考超級嬰兒通，目的是讓小 Baby 入睡時仍知道是在床上入睡，而不是養成在大人懷抱入睡的習慣。雖然看著 Baby 在身上快睡著時，要馬上放回去 … 真的有挑戰（怕她又大哭），可是大概來回幾次（哭了又抱起，快睡了又放下） … Baby 也真的會入睡了 … 重點是會睡的很不錯，接下來就一覺到天亮了。

每一次我看到云云熟睡的臉，以及起床快樂又充滿精神的樣子。我就覺得我們（我&JJ）花這些時間，調整好她的睡眠習慣 … 實在是非常值得啊！重點是云云開心（睡的好睡的飽），我們大人（也一樣睡的好睡的飽）也開心啊！

不過在過程中，還是會有一些挑戰的，也會有許多機會考驗父母的信心 … 記得多看一些書，找一些自己接受的方法，來測試一下 … 相信一陣子後，自然會有心得的！

也希望您的 Baby，一樣睡的好 ^_^
