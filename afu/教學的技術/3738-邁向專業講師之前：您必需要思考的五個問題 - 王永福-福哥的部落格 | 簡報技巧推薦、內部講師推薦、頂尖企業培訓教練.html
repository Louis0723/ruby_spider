坐在電腦前上facebook，訊息對話框突然跳了出來：「我想當講師，不知道福哥覺得合適嗎？」原來是朋友 Danny 最近工作上有些阻礙，看到講師的工作似乎蠻吸引人的，因此興起了轉行當專職企業訓練講師的念頭。幾年前我曾是他內部講師課程教練，在演練時他有很不錯的表現，因此這個念頭一起來，他第一個想到向我詢問。

身為專業的企業訓練講師（或稱為職業講師），我真心的喜歡我現在的角色（或者稱為工作、職業、事業，或志業）。講師有舞台、有相對自由的時間、有不錯的收入、還能獲得學員的尊敬 … 怎麼看都像是個理想的工作。

面對Danny，我並不想故意把這個行業講的很可怕。我熱愛這份職業，也以身為講師為榮。只是從台下看台上，常常看到的只是光環，卻沒有看到光環背後付出的代價，我不想 Danny 過度樂觀，沒有作好評估就一頭熱投入。只希望以過去幾年來在講台上的經驗，真實而中性的提出幾個問題，讓他可以自我評估適不適合。

 

當然我的意見，僅代表一個參考的方向，雖然現在大部份的時間，我都在上市公司：包含鴻海、台積電、西門子、中信金等公司，擔任外聘訓練講師的工作，但在踏入這個行業的一開始，我也曾作過課程開發 / 講師安排 / 公開班招生 / 校園演講 / 公開講座 … 等不同型態的講師。曾經摸索過，也曾經低潮過。我只是把一些想法整理一下，也提供給想要踏入講師這一行的朋友，一個參考的方向。以下五件事，是您在邁向專業講師之路前，必需要思考的五個問題：

 

1.你能堅持多久

讓我們面對現實：我知道您很行！但是別人不知道，別的單位不知道，別的公司不知道。在您真的站穩舞台前 … 您可能會有很長的一段時間 … 不知道下一堂課在那裡！就如同一個剛出道的歌星或演員，有能力並不代表有演出的機會。極少部份的人可能很快就紅了，但更多的是苦等下一個演出的機會，或只領微薄的通告費。到底什麼時候您會發光發熱 … 您不知道，我不知道 … 沒人會知道！

因此在您真正站穩專業講台之前，我想問的一個問題是：「您能堅持多久？您打算堅持多久？」這裡指的不止是心理及精神上的堅持，更重要是實質及財務上的堅持。以我自己為例：我在剛開始踏入講師這個領域時，常常一整個月 … 一堂課都沒有！是的！那表示整個月的收入是：零！！最慘的時候，戶頭裡剩不到 1000（不誇張，沒有故意裝可憐…）。這樣的狀態整整持續了三年！

國內最知名的企業講師，也是暢銷書作家－憲哥說過一句話：「講師有二種：一種是餓死的講師，一種是累死的講師」。我十分認同這個說法！在您真的站穩講台邀約不斷之前，您打算在舞台下堅持多久？您的存糧又能讓您堅持多久（而不餓死？）。在您真正選擇講師成為您的職業之前，這點是非常現實，也非常重要的評估！

 

2.您想要如何開始？

下一個問題是：「您打算如何開始您的講師事業？」白話的說：就是您如何行銷您自己？以及您的課程？

您不是張惠妹或謝金燕，沒有人會主動來邀約您去表演的！還是實際一點 … 先想想好怎麼開始吧？不論是自我開發 / 透過管顧引介 / 公開班招生 / 接校園演講 / 出書建立知名度 / 與雜誌專欄合作 / 經營專業社群 / 加入同好社團 / 寫 Blog 或網站 / 在實體或網路打廣告 … 上面這些方法都可以找到成功的案例。重點是：您打算怎麼作？

其比較危險的是您沒有方向，又過度樂觀，以為您只要打幾通電話，或是發幾封 E-mail … 就會有雪花一般的邀約。也許您真的是如此，但如果沒有呢？當然，我也沒有一個絕對可行的方向。唯一有的是：我會去嘗試！我請過助理作電話開發，也找過外包的電話行銷公司；我寫了部落格文章，也下過網路廣告；我曾自行開班招生，也自行接觸不同的管顧顧問公司或課程經紀人 …。到底那一個對您效果最好 … 我無法給您答案，但我一定可以確定的是：您必需要親自試過後，才真的知道。

重點是：您想過這些事嗎？您想過開始的方向嗎？

 

3.您 Hold 的住場面嗎？

身為一個職業講師，您所面對的常常是一群帶著既有經驗（或是成見）的企業人士，他們來到您的課堂上，經常是帶著評估的角度，先評估講師是不是夠有料？教的東西對我有沒有用？能不能吸引我？等到先信服後，才會有學習及改變的產生。

當您站在台上，面對帶有懷疑的目光，您能不能在 5 分鐘內 Hold 住場面，不止您對自己有信心，還能讓學員對您有信心。這不僅是專不專業的問題，而是挑戰您的授課及技巧的問題。有時教到一半，也可能會遇到學員對您的教學內容提出質疑，或採取不配合的態度，您又該如何處理？這些都是很實際，而且經常會面對到的狀況，您是否有信心可以處理？

更進一步是：您的授課內容有什麼獨到之處，是足以讓人信服 & 佩服的？請記得您面對的不是大學生（雖然大學生有時更難教 ^^），而是身經百戰的職場人士（當然有時也可能是職場菜鳥）。您有沒有料 … 其實大概一節課就會知道。在每一堂課結束後，你馬上就會被評價－也就是課程滿意度的評價。如果您前幾次的表現不夠「傑出」，那也就沒有接下來了機會了。曾經聽過 HR 告訴我：如果新聘講師滿意度不到 4.3（滿分5分），那麼她們就不會再邀請這個講師。這對新講師絕對會是一個挑戰。

所以，不論面對誰，您有沒有信心及能力 Hold 住現場（而且持續 Hold 住，有時長達一整天），有沒有能力讓大家信服！這也是您應該要問自己的，也真實評估的一個問題。

 

4.您的體力能否負荷？

一般職業講師或企業內訓講師，常常一站就是一整天，整整 7 個小時的時間站在講台前，由您一個人獨撐一整天，這對體力真的是一個挑戰。如果遇到有時連續排課，不管是連續 3 天，或連續 4 天每天站著，對腳及腰都會造成很大的壓力。而像是聲音的要求，即使累了都不能聽起來覺得累，啞了也還是得大聲講話，要保持一整天的活力及上課的熱情。這些都是對職業講師必備的的要求！

嗯 … 這麼說好了 …. 偶爾去大魯閣打打棒球，是蠻休閒也蠻快樂的。但如果身為職業選手要上場揮棒，那可就完全是不同等級的要求了！

我們都還沒提到講師必需提早出門提早抵達：標準 9 點開始的課，一定在 8:30 就會到現場準備，不論是台北 / 新竹 / 台南 / 高雄 … 都是 8:30 要到！那您可以回推一下，您應該要幾點起床，才會來得及趕上高鐵（或其它的交通工具）。然後還要上課站一整天，然後再回到家裡 … 說不定明天還繼續有課。有些老師還得出差去大陸教課，在大陸不同的城市中移動！！這些都需要時間，也都需要體力支持 … 。

曾經有一次跟幾位知名講師齊聚，聊的不是教課的內容或奇聞異事，卻都在聊：那一位講師在台上站到受傷 … 那一位講師生病後還在台上連站3天 … 又有那一位講師出院後馬上回到講台上 … 這些都是在講台下所看不到的部份。也是您應該納入自我評估的部份。

 

5. 您是否有足夠的教學熱情？

我們假設您順利通過了考驗，讓市場接受您的課程。逐漸的累積您的口碑及知名度，有越來越多的邀約出現，然後您不斷上台講課。這時您可能不再有會不會餓死的問題，反而開始出現會不會累死的問題（還記得憲哥的名言：「講師只有二種，一種是累死的講師，一種是餓死的講師」。）

一個值得問自己的問題是：「同樣的課連續上了100場，您上台是否還有熱情？」

講師也是人，也都會累，也會有個人的情緒及困擾。當您站上講台（可能是您這個月第 15 場，今年第100場），面對學員時（然後您看到學員有人因為是被長官指派來上課而臭臉），相信我 … 支持您站在講台上的，會是您的教學熱情！您是否真的有熱情想跟人分享？您是否真的有熱情站講台上？最終這會是一個決定的關鍵！因為這份熱情，才會趨使您更往前進。因為這份熱情，才會幫助您精益求精。因為這份熱情，才會幫助您成為一個炙手可熱，大家爭相邀約的一流講師。

更重要的：也因為有這份教學的熱情，您才真的可以樂在其中，接受所有的挑戰，並以專職的企業訓練講師為榮！

 

後記

跟 Danny 見面時，我請他評估一下，如果真的要選擇走講師這條路 … 自己能夠堅持多久？想要怎麼開始？能否 Hold 住現場？體力狀況如何？以及是否真的有教課的熱情？ … 我並沒有給他任何答案，只把問題丟給他自己評估。也許 … 在問自己問題的過程裡，答案早已浮現在您的心中 … 只有您自己最清楚。

祝福您！我的朋友！
