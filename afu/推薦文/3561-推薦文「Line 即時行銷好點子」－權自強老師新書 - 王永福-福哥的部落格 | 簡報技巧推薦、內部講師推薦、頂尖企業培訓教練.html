平常在台灣各大上市公司擔任顧問時，有時總是會聽到客戶抱怨：不知道要怎麼運用社群及網路通訊技術，來讓他們產品的銷量可以更好？雖然工具不少，卻不曉得要怎麼用好？

這些問題，相信各位會從權自強老師的新書中，得到充份而完整的解答。

僅僅只是翻閱書中第一個案例：「牛肉麵」運用 Line 行銷實例，對我就有很大的啟發。原來傳統產業，也可以用 Line 進行低成本，甚至無成本行銷啊！

 

書裡面也仔細的說明，如何運用 Line 去接觸客戶、黏住客戶、最後成交客戶，再變成滿意的客戶。我才知道原來看似簡單的 Line 即時通訊軟體，還可以發揮這麼大的效用。權老師在書上也提醒大家，應該注意那些常見的問題，甚至細到如何發訊息？如何增加粉絲，書裡都有詳細的操作方法。

這本書也是我看過關於 QR Code 在行銷運用實務，最完整也最清楚的一本書。書裡舉了許多國內外不同的 QR Code 運用實例，讓讀者能一眼就清楚，未來如何運用 QR Code 在自己產品的行銷領域。真的非常完整而有用。權老師在書上更無私的分享：許多他們為不同品牌公司操作的經驗。這其實不僅是一本工具書，對於許多正在開拓客戶，經營事業的讀者們，這更是一本網路行銷求生手冊。我衷心的推薦本書，相信大家看完後會有很的收獲！
