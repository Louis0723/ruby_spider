身為講者，我就不要再寫心得 …. 來寫一些背後大家可能不知道的事吧？

問：什麼是「最小化資訊要求、最大化教學效果」？

答：這是我在設計這五場「線上教學的技術」時，最主要的核心策略。我希望突破線上教學的資訊障礙，老師不用被太多的資訊工具所綁住，學生也不需要卡在工具之間的不斷轉換。我想要用最簡單、最直覺、甚至是大家最熟悉的方法－也就是紙跟筆，來完成一場好的且有效的線上教學！

問：是不是因為福哥對資訊技術或新工具不熟，才不去用這些軟體？

答：嗯，剛好相反！我玩電腦超過30年了！（第一台電腦 8088），用 PE2 + HTML 手寫出國內第一個建設公司的網站、寫了國內第一本 Joomla 123 架站很簡單的 CMS 電腦工具書，現在讀的也是資管博士的學位 … 我太喜歡電腦跟資訊工具了啊！

問：那為什麼線上教學要「最小化資訊要求」，不去用很多好用的線上工具，如 Kahoot、Quizlet、不同軟體的線上電子白板（如 Google Jamboard、或 Zoom 的電子白板）

答：說在前面，我覺得像 Kahoot 、Quizlet 等都很棒啊！我還有 Quizlet 的付費帳號，幫孩子背英文單字。但身為老師，每一個軟體的使用，就是一個工具的門檻（總是要先摸索學習一下，這個要花時間）；而每一個軟體的切換，又是一個注意力的轉換（切過去、再切回來，你確定學生回來的那麼快？）。一直追求資訊工具的使用，對老師及學生有可能會產生工具學習的負擔甚至焦慮，最後反而會讓大家忘了教學跟學習的本質。

問：那什麼是「教學的本質？」

答：這個問題很大，我們先聚焦在一個很小的範圍就好。那就是「請問你的課程精彩嗎，能吸引學生？讓學生有收獲嗎？」。我們先不要管實體或線上，就只單問這個核心！如果你的課程精彩，能夠吸引學生，那接下來要面對的只是從實體到線上轉換的問題！如果你的課程本來就有點乾 …（當然，也可能是有太多乾貨 … XD），那不管是實體或線上，也都會有點乾。教學時有些核心，不因為線上／線下，而有所轉變，這是我所說的「教學的本質」

問：可以舉一些實例，說明「最小化資訊要求、最大化教學效果」嗎？

答：這幾次「線上教學的技術」的課程或演講，我只要求大家準備白紙跟一支筆，就可以讓大家用手邊的紙筆參與討論，像是選擇或排序法（在紙上寫答案，Show 在螢幕上）、問答法（實體在螢幕前舉手、人數太多時才開放虛擬舉手）、小組討論法（讓大家分為小組討論，並且實際寫下答案在手邊白紙，秀在螢幕前跟大家分享）、演練法（依造我在螢幕前的指示跟示範，由學員自己完成一些動作，並且不會出錯）。還能用最簡單的工具，完成從小班到規模 200 人的大型互動演講。從實際的操作中証明，不用過多的資訊工具，還是能完成一場場精彩的課程或演講。

問：那就不應該用資訊工具了嗎？

答：啊，並不是！更精準的說：不要被資訊工具所限！不要因為卡在資訊工具前，卻忘了教學或學習的本質！如果老師們心裡已經規劃好，覺得課程中就應該導入某一個資訊工具，才會讓學習效果更好！也經過仔細考量，知道如何在課前或課中，讓學生熟悉工具使用，並同步完成進度確認，最後也思考到：不同工具切換的時間，以及如何把學生叫回來，並且再度回到課程。在這些考慮清楚的狀態下，我認為不論是資訊工具，或是非資訊工具，只要對教學效果跟學習成效有幫助，都是最好的選擇！

問：線上教學除了課程本身，還有哪些是重要的關鍵？

答：這次「線上教學的技術」200 人大場演講，除了教學跟演講的內容設計之外。最大的挑戰是課前連絡的即時性管道建立！從週六早上葉丙成老師發佈訊息，到週日早上 11:00 開始演講，我們只有大約24小時的時間，從報名、挑選、通知、上課，全部要完成！在這邊要謝謝憲福育創（我跟憲哥的公司）兩位夥伴：Ariel ＆ Emory 的線上行政支援，不管入選或沒入選，都一一的回覆 Email 給大家，也謝謝合夥人憲哥總是支持我的任性XD；謝謝 Andy 劉滄碩老師對 Line 群組訊息的顧問指導，讓大家可以快速成立一個即時課程訊息通知社群！不僅讓我可以發佈課前提醒、教具準備、以及時間及線上教室地點，甚至還在緊急時發揮了超重要的功能。也建議老師們，可以想一下，如何建立課程專用的連絡管道，有問題時才能即時通知＆解決。（其實我很少用 Line，而這個課程 Line 訊息也請大家不要聊天或打招呼，只為課程連絡用。當然，用Email 也行！）

問：線上課程有遇到什麼突發事故嗎？如何解決？

答：現在想起來還驚魂未定！（先讓我駡一聲 OOXX !）。最可怕的是週日 200 人場，我們提早 20 min 開會議室，結果在開始前 10 分鐘，會議室卡住了！大家開始進不來！我從即時訊息 Line 那裡發現這個狀況！心跳停了一拍！然後開始緊急處理！原來是 Zoom 的付費版，還分成不同人數！一般的付費版最大上限是 100 人！我沒 Run 過這麼大的線上即時場（這個不是一般的直播，而是每個人都要開鏡頭互動的！），沒有注意到這件事 … ! 趕快手忙腳亂的進行線上昇級，費用都沒看卡就給他刷下去了！還好可以即時昇級！嚇死我！

然後，70 人的測試場，則是發生網路太慢造成即時影像大 Delay！結果是因為 Mac 筆電自己選擇我家的舊無線網路！而不是新的 Mesh AP！還好我手邊有一個監看用的設備，即時發現後馬上處理，也有驚無險！

所以，線上課程還是有很多不同的檢查點！要遇過才知道啊！（還好哥哥有練過，不要嚇我！）

問：為什麼這麼好？會免費分享「線上教學的技術」？背後有什麼動機嗎？

答：過去我早已經把職業講師在上市公司的訓練手法，寫成教學的技術系列文章，在福哥的部落格免費公開；2019 年則集結文章成書，出版「教學的技術」一書；2020年則用突破性的三機三鏡手法，推出「教學的技術－線上課程」，同時也打破台灣線上課程那時的募資銷售記錄，有超過5000位老師購買。這次202105疫情突然來襲，全國的老師們瞬間被迫轉變為線上教學，我在 5/18 早上（下午才發佈全國停課），寫下「發展遠距教學的技術 …. 我不出手，誰能 … 」（請原諒我自我激勵的話語 XD），只希望提供一些個人經驗跟角度，跟有需要的老師們一起分享。也許讓更多人知道「教學的技術」，知道這些技術可以突破線上跟線下，然後讓更多老師認識我 … 算是對我的好處吧？XD

問：這幾次「線上教學的技術」參與者是怎麼挑的，為什麼都沒挑中我？

答：先講最大場 200 人的！因為報名場超過 2000 人！（大驚！），我們覺得目前在學校任教的老師，應該最需要這樣的火力支援！但人數還是太多，所以我們是用電腦隨機數產生器來挑選。而12、15、25 人的測試場，我當然找熟人夥伴，也就是參加過我實體課程－包含專業簡報力、教學的技術、以及憲福育創的學員，一起來參加測試場（這樣表現失常或要調整時，他們都能接受）。而 70 人，則是大部份為完全不熟的學校老師、少部份熟人（我這個人還是胳臂往內彎 XD）。But Anyway ，總是沒參加的人還是很多，在此跟大家致歉！福哥也會繼續寫文章，或想想有沒有什麼其它的方法，跟大家分享的！

問：所以，是要大家去買「教學的技術」或「教學的技術－線上課程」嗎？

答：請勿衝動！把信用卡收起來！XD。「教學的技術」全書裡至少有 80 ％ 的內容，在福哥的部落格都找的到！免費的！「教學的技術－線上課程」的精華，也就「小組討論法」及「遊戲化」，也已經免費釋出在網路及福哥的部落格上！然後我之前說過，因為合約關係，「教學的技術－線上課程」已經調回原價 6800 （雖然相對於實體課程的六位數價格 … 但疫情的現在，這個價格對很多老師仍是負擔）。希望大家先不要買 … ！我們再來看看能夠怎麼做好嗎？

問：請問還會有免費的演講或課程體驗嗎？

答：為了週日早上的 200 人規模演講，我先試 Run 了三場教學，從 12人、25人、然後再換 Webex 做了15人。而演講前整個週六更是自我隔離備課（從早上9:00 到晚上 7:00），然後週六晚上再辦一場70人測試場。測完之後再課後檢討 AAR ＆即時修正到淩晨 1點半。隔天早上再起來準備200人大場 …。然後整個假日真的像拋家棄子。真是累到爆炸 …。但看到大家滿滿的心得跟收獲，十分滿足！但現在手邊還有一些專案：像是簡報認證推動＆簡報溝通協會網站、教學的技術－線上課程週年慶專案、還有明年本來就計劃的教學遊戲化的技術專案，甚至是我還繼續等待中的博士論文及期刊回覆 … 。這些都停了2週了 XD。讓我先喘一下，清一下手邊的工作，再來看看後續的可能性。

然後，再次謝謝大家，我仍然會持續整理，持續寫作「線上教學的技術」系列文的！也歡迎你訂閱哦！
